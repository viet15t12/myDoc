01. Cài đặt Go và chương trình đầu tiên

Bài mở đầu giúp bạn thiết lập môi trường Go, tạo module đầu tiên và chạy chương trình bằng go run.

Mục tiêu

Mục tiêu bài học: cài Go, hiểu module ở mức cơ bản, viết file main.go và chạy chương trình đầu tiên.
  • Biết Go phù hợp với backend, CLI, network service và hệ thống hiệu năng cao.
  • Cài Go và kiểm tra phiên bản.
  • Tạo module và chạy file main.go đầu tiên.

1. Go là gì?

Go là ngôn ngữ biên dịch, cú pháp gọn, tooling mạnh và hỗ trợ concurrency tốt. Điểm hấp dẫn của Go là tốc độ build nhanh, binary dễ triển khai và thư viện chuẩn thực dụng.

2. Cài đặt

Sau khi cài Go, kiểm tra trong terminal:

go version

Nếu lệnh trả về phiên bản Go, môi trường đã sẵn sàng.

3. Tạo module

Module là đơn vị quản lý mã nguồn và dependency trong Go.

mkdir hello-go
cd hello-go
go mod init example.com/hello-go

4. Hello world

Tạo file main.go:

package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}

Chạy chương trình:

go run .

5. Lệnh cần biết

go fmt ./...
go test ./...
go build
go run .
Lưu ý: chạy go fmt ./... thường xuyên để code Go giữ đúng style chuẩn của hệ sinh thái.

Tổng kết

Bạn đã có module Go đầu tiên. Bước tiếp theo là học biến, kiểu dữ liệu, function và cách Go xử lý lỗi.
