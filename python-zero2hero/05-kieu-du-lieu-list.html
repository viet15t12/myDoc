05. Kiểu dữ liệu List trong Python

Tài liệu học Python cơ bản về kiểu dữ liệu list: tạo list, truy cập theo index, slicing, duyệt phần tử, nối list và các built-in methods quan trọng như append, extend, sort, insert, pop…

Giới thiệu

Python có bốn kiểu dữ liệu dùng để lưu collection of data (nhiều giá trị cùng lúc):

Bài này tập trung vào list — kiểu được dùng nhiều nhất. List là kiểu mutable: sau khi tạo, bạn hoàn toàn có thể thay đổi, thêm, xóa từng phần tử. Đây là điểm khác biệt lớn so với string (immutable) đã học ở bài 03.

1. Tạo list

Dùng cặp dấu ngoặc vuông []. Phần tử trong list có thể là bất kỳ kiểu dữ liệu nào — thậm chí là một list khác. 

my_list = [1, 3.5, 'string', [5, 7]]

print(my_list)
print(type(my_list))

Kết quả:

[1, 3.5, 'string', [5, 7]]
<class 'list'>
Ghi nhớ: list có thể chứa hỗn hợp các kiểu dữ liệu khác nhau trong cùng một list. Phần tử [5, 7] ở trên là một list lồng bên trong (nested list).

2. List trống

Có hai cách tạo list trống:

# Cách 1: dùng dấu ngoặc vuông rỗng
empty1 = []

# Cách 2: dùng hàm list()
empty2 = list()

print(empty1)   # []
print(empty2)   # []
Khi nào dùng? List trống thường được tạo trước, rồi thêm dần phần tử vào trong vòng lặp bằng append().

3. Chiều dài list

Hàm len() trả về số phần tử trong list — tương tự như với string.

my_list = [1, 2, 3, 4]
print(len(my_list))   # 4

4. Truy cập phần tử theo index

Cú pháp giống string: dùng dấu ngoặc vuông []. Index dương bắt đầu từ 0; index âm đếm từ cuối (-1 là phần tử cuối).

Phần tử13.5'string'[5,7]
Index dương 0123
Index âm -4-3-2-1 
my_list = [1, 3.5, 'string', [5, 7]]

print(my_list[1])    # 3.5  — phần tử thứ hai (index 1)
print(my_list[-1])   # [5, 7] — phần tử cuối cùng

Kết quả:

3.5
[5, 7]

5. Slicing – lấy một đoạn phần tử

Cú pháp slicing của list hoàn toàn giống string: list[start:stop] — lấy từ start đến stop - 1. 

my_list = [1, 3.5, 'string', [5, 7]]

# Lấy từ index 1 đến cuối
print(my_list[1:])    # [3.5, 'string', [5, 7]]

# Lấy từ đầu đến index -2 (không bao gồm -1)
print(my_list[:-1])   # [1, 3.5, 'string']

# Lấy từ index 1 đến index -2 (không bao gồm -1)
print(my_list[1:-1])  # [3.5, 'string']

Kết quả:

[3.5, 'string', [5, 7]]
[1, 3.5, 'string']
[3.5, 'string']

6. Duyệt qua các phần tử

Dùng vòng lặp for để duyệt qua từng phần tử:

my_list = [1, 3.5, 'string', [5, 7]]

for x in my_list:
    print(x)

Kết quả:

1
3.5
string
[5, 7]
Lưu ý cú pháp: dòng print(x) phải được thụt vào (indent) so với for để Python hiểu đó là phần thân của vòng lặp.

7. Nối list

Dùng toán tử + để ghép hai list thành một list mới — tương tự nối chuỗi.

list1 = [1, 4.5, 10]
list2 = ['a', 'b']

concat = list1 + list2
print(concat)

Kết quả:

[1, 4.5, 10, 'a', 'b']
Chú ý: toán tử + tạo ra một list mới, không thay đổi list1 hay list2. Nếu muốn thêm phần tử vào list gốc, dùng extend() (xem mục 10).

8. Thay đổi phần tử — list là mutable

Khác với string (immutable), list cho phép gán lại giá trị tại một vị trí index bất kỳ: 

my_list = [1, 4.5, 10]
my_list[1] = 'hello'   # thay phần tử ở index 1
print(my_list)

Kết quả:

[1, 'hello', 10]
So sánh với string: s[1] = 'x' trên string sẽ báo TypeError. Với list, thao tác tương tự hoàn toàn hợp lệ vì list là mutable.

9. append() – thêm phần tử vào cuối

append(value) thêm value vào cuối list. Đây là method được dùng nhiều nhất khi làm việc với list. 

my_list = [1, 3, 6, 10, -5]
my_list.append(50)
print(my_list)

Kết quả:

[1, 3, 6, 10, -5, 50]
Thực tế: append() thường được dùng trong vòng lặp để xây dựng list dần dần, ví dụ thu thập kết quả tính toán, lọc dữ liệu từ nguồn khác…

10. extend() – mở rộng list bằng iterable

extend(iterable) thêm tất cả phần tử của một iterable (list, tuple, string…) vào cuối list gốc. Khác với +, extend() thay đổi trực tiếp list gốc thay vì tạo list mới. 

my_list = [1, 3, 6, 10, -5]
list1   = ['anh', 'em']

my_list.extend(list1)
print(my_list)

Kết quả:

[1, 3, 6, 10, -5, 'anh', 'em']

So sánh append vs extend

Khác biệt quan trọng giữa append() và extend()
MethodThêm cái gì?Ví dụKết quả
append([4,5]) Thêm cả list như một phần tử [1,2].append([4,5]) [1, 2, [4, 5]]
extend([4,5]) Thêm từng phần tử riêng lẻ [1,2].extend([4,5]) [1, 2, 4, 5]

11. sort() và sorted() – sắp xếp

sort() — sắp xếp tại chỗ

Sắp xếp các phần tử của list ngay trên list gốc (in-place). Mặc định tăng dần. 

my_list = [1, 3, 6, 10, -5]

my_list.sort()               # tăng dần
print(my_list)               # [-5, 1, 3, 6, 10]

my_list.sort(reverse=True)   # giảm dần
print(my_list)               # [10, 6, 3, 1, -5]

sorted() — trả về list mới

Không thay đổi list gốc, trả về một list mới đã sắp xếp. 

my_list  = [1, 3, 6, 10, -5]
new_list = sorted(my_list)

print(my_list)    # [1, 3, 6, 10, -5]  — không đổi
print(new_list)   # [-5, 1, 3, 6, 10]  — list mới đã sắp xếp
Chọn sort() hay sorted()
sort()sorted()
Thay đổi list gốc?CóKhông
Trả về list mới?Không NoneCó
Dùng khi nào?Muốn sắp xếp luôn list gốcMuốn giữ nguyên list gốc

12. reverse() và [::-1] – đảo ngược

reverse() — đảo tại chỗ

my_list = [1, 3, 6, 10, -5]
my_list.reverse()
print(my_list)   # [-5, 10, 6, 3, 1]

[::-1] — tạo list mới đảo ngược

my_list  = [1, 3, 6, 10, -5]
new_list = my_list[::-1]

print(my_list)    # [1, 3, 6, 10, -5]  — không đổi
print(new_list)   # [-5, 10, 6, 3, 1]  — list mới
Tương tự string: [::-1] hoạt động với cả string lẫn list theo cùng nguyên lý slicing với step -1.

13. insert() – chèn phần tử vào vị trí chỉ định

insert(index, value) chèn value vào vị trí index, đẩy các phần tử phía sau về bên phải. 

my_list = [1, 3, 6, 10, -5]

my_list.insert(0, 100)   # chèn 100 vào đầu (index 0)
print(my_list)           # [100, 1, 3, 6, 10, -5]

my_list.insert(1, 200)   # chèn 200 vào vị trí thứ hai (index 1)
print(my_list)           # [100, 200, 1, 3, 6, 10, -5]

14. del – xóa phần tử theo index

Dùng lệnh del để xóa phần tử tại một index cụ thể: 

my_list = [1, 3, 6, 10, -5]

del my_list[2]   # xóa phần tử tại index 2 (giá trị 6)
print(my_list)   # [1, 3, 10, -5]

Xóa nhiều phần tử bằng slicing

my_list = [1, 3, 6, 10, -5]

del my_list[:2]   # xóa hai phần tử đầu tiên (index 0 và 1)
print(my_list)    # [6, 10, -5]

15. index() – tìm vị trí của phần tử

index(value) trả về index của lần xuất hiện đầu tiên của value. Nếu không tìm thấy, báo ValueError. 

my_list = [1, 3, 6, 10, -5]

idx = my_list.index(10)
print(idx)   # 3
Lỗi khi phần tử không tồn tại: 
my_list.index(100)
# ValueError: 100 is not in list
Để tránh lỗi, kiểm tra trước bằng in: 
if 100 in my_list:
    print(my_list.index(100))
else:
    print("Không tìm thấy")

16. pop() – lấy và xóa phần tử

pop(index) vừa xóa phần tử tại index khỏi list, vừa trả về giá trị của phần tử đó. Nếu không truyền index, mặc định xóa phần tử cuối cùng. 

my_list = [1, 3, 6, 10, -5]

element = my_list.pop(3)   # xóa phần tử tại index 3 (giá trị 10)
print("Phần tử bị xóa:", element)   # 10
print("List còn lại:", my_list)     # [1, 3, 6, -5]

Kết quả:

Phần tử bị xóa: 10
List còn lại: [1, 3, 6, -5]
So sánh pop() vs del: cả hai đều xóa phần tử theo index, nhưng pop() trả về giá trị đã xóa còn del thì không.

19. Bài tập thực hành

Bài tập 1: Xây dựng list từ input

Nhập 5 số nguyên từ bàn phím, lưu vào một list, sau đó in ra list, chiều dài, phần tử lớn nhất và nhỏ nhất.

numbers = []
for i in range(5):
    n = int(input(f"Nhập số thứ {i+1}: "))
    numbers.append(n)

print("List:", numbers)
print("Lớn nhất:", max(numbers))
print("Nhỏ nhất:", min(numbers))

Bài tập 2: Lọc số chẵn

Cho list [3, 8, 1, 12, 7, 4, 9, 6]. Tạo một list mới chỉ chứa các số chẵn.

numbers = [3, 8, 1, 12, 7, 4, 9, 6]
even = []
for n in numbers:
    if n % 2 == 0:
        even.append(n)
print(even)

Bài tập 3: Xóa trùng lặp

Cho list [1, 2, 2, 3, 4, 3, 5]. Tạo một list mới không có phần tử trùng lặp, giữ nguyên thứ tự xuất hiện đầu tiên.

Bài tập 4: Đảo ngược list thủ công

Viết code đảo ngược list mà không dùng reverse() hay [::-1] — chỉ dùng vòng lặp.

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • List là kiểu mutable, có thứ tự, cho phép trùng lặp, có thể chứa hỗn hợp kiểu dữ liệu.
  • Tạo list bằng [] hoặc list(). List trống: [].
  • Truy cập phần tử bằng index dương / âm; lấy đoạn bằng slicing.
  • Gán lại list[i] = v hợp lệ vì list là mutable — khác với string.
  • append(v) thêm vào cuối; extend(iter) thêm nhiều phần tử từ iterable.
  • sort() sắp xếp tại chỗ; sorted() tạo list mới.
  • insert(i, v) chèn vào vị trí; del list[i] / remove(v) xóa phần tử.
  • pop(i) xóa và trả về phần tử — hữu ích khi cần cả xóa lẫn lấy giá trị.
Bài tiếp theo: Tuple — kiểu dữ liệu tương tự list nhưng immutable. Hiểu rõ list sẽ giúp bạn học tuple nhanh hơn nhiều vì cú pháp rất giống nhau.
