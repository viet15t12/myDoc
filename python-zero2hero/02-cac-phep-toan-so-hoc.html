02. Các phép toán số học trong Python

Tài liệu học Python cơ bản, tập trung vào các phép toán số học: cộng, trừ, nhân, chia, chia lấy dư, lũy thừa, floor division và toán tử gán kết hợp.

Giới thiệu

Sau khi đã biết biến và kiểu dữ liệu, bạn có thể dùng Python để thực hiện các phép tính trên số nguyên int và số thực float. Các phép toán số học là nền tảng cho bài toán tính tiền, tính điểm, chuyển đổi đơn vị, xử lý dữ liệu và rất nhiều chương trình thực tế.

Mục tiêu bài học: biết cú pháp các phép toán số học, hiểu sự khác nhau giữa /, //, % và biết áp dụng vào ví dụ thực tế.

1. Bảng phép toán số học trong Python

Phép toán Ký hiệu Ví dụ Kết quả
Cộng + 3 + 10 13
Trừ - 3 - 10 -7
Nhân * 3 * 10 30
Chia / 10 / 3 3.333...
Chia lấy dư % 10 % 3 1
Lũy thừa ** 2 ** 4 16
Chia lấy phần nguyên dưới // 10 // 3 3

2. Phép cộng +

Phép cộng dùng để cộng hai giá trị số.

print(3 + 10)

Kết quả:

13

3. Phép trừ -

Phép trừ dùng để lấy số bên trái trừ đi số bên phải.

print(3 - 10)

Kết quả:

-7

4. Phép nhân *

Phép nhân dùng dấu sao *, không dùng ký hiệu x.

print(3 * 10)

Kết quả:

30

5. Phép chia /

Phép chia bằng dấu / luôn trả về kết quả kiểu số thực float, kể cả khi chia hai số nguyên. 

print(10 / 3)
print(type(10 / 3))

Kết quả:

3.3333333333333335
<class 'float'>
Ghi nhớ: 10 / 3 là chia thông thường, kết quả có phần thập phân.

6. Ép kiểu kết quả chia về số nguyên

Nếu muốn lấy phần nguyên của kết quả chia dương, bạn có thể dùng int() để ép kiểu. 

result = 10 / 3
print(result)
print(int(result))

Kết quả:

3.3333333333333335
3
Chú ý: int() không giống hoàn toàn // khi làm việc với số âm. Ví dụ int(-10 / 3) ra -3, còn -10 // 3 ra -4.

7. Phép chia lấy dư %

Phép chia lấy dư trả về phần dư của phép chia.

print(10 % 3)

Kết quả:

1

Với số âm, kết quả có thể khác cách nghĩ thông thường nếu chỉ quen chia tay:

print(-10 % 3)

Kết quả:

2
Giải thích nhanh: Python giữ quan hệ n = d * (n // d) + (n % d). Vì -10 // 3 = -4, nên -10 = 3 * (-4) + 2.

8. Phép lũy thừa **

Phép lũy thừa dùng hai dấu sao **.

print(2 ** 4)

Kết quả:

16

Có thể dùng số thực làm số mũ:

print(2 ** 4.5)

Kết quả xấp xỉ:

22.627416997969522

9. Floor division //

Floor division là phép chia lấy số nguyên lớn nhất nhỏ hơn hoặc bằng kết quả chia thật. Có thể hiểu đơn giản là phép chia làm tròn xuống. 

print(10 // 3)

Kết quả:

3

Với số âm:

print(-10 // 3)

Kết quả:

-4
Dễ nhầm: -10 / 3 xấp xỉ -3.333. Số nguyên lớn nhất nhỏ hơn hoặc bằng -3.333 là -4, không phải -3.

10. Công thức liên hệ giữa // và %

Trong Python, phép chia nguyên và phép chia lấy dư có quan hệ:

n = d * (n // d) + (n % d)

Ví dụ với n = 20, d = 3:

n = 20
d = 3

floor_division = n // d
remainder = n % d

print("Floor division:", floor_division)
print("Remainder:", remainder)
print(d * floor_division + remainder)

Kết quả:

Floor division: 6
Remainder: 2
20

11. Toán tử gán kết hợp

Khi muốn lấy giá trị hiện tại của biến để tính tiếp rồi gán lại vào chính biến đó, bạn có thể viết ngắn gọn bằng toán tử gán kết hợp.

Cách viết thông thường

number = 10
number = number + 5
print(number)

Cách viết ngắn gọn

number = 10
number += 5
print(number)

Cả hai đoạn đều cho kết quả:

15
Toán tử Tương đương Ý nghĩa
x += 5 x = x + 5 Cộng thêm 5
x -= 5 x = x - 5 Trừ đi 5
x *= 5 x = x * 5 Nhân với 5
x /= 5 x = x / 5 Chia cho 5
x %= 5 x = x % 5 Lấy dư khi chia cho 5
x //= 5 x = x // 5 Chia lấy phần nguyên dưới
x **= 5 x = x ** 5 Nâng lên lũy thừa 5

12. Độ ưu tiên toán tử

Python tính toán theo thứ tự ưu tiên. Nếu muốn chắc chắn thứ tự tính, hãy dùng dấu ngoặc tròn ().

Mức ưu tiên Toán tử Ví dụ
Cao ** 2 ** 3
Trung bình *, /, //, % 2 + 3 * 4 → 14
Thấp +, - (2 + 3) * 4 → 20

13. Ví dụ thực hành: Tính tiền đi chợ

Bài toán: giá rau là 15.000đ/kg, giá thịt là 120.000đ/kg. Mua 2kg rau và 0.5kg thịt. Hỏi tổng số tiền là bao nhiêu? 

gia_rau = 15000
gia_thit = 120000

tong_so_tien = 2 * gia_rau + 0.5 * gia_thit

print("Tổng số tiền đi chợ:", tong_so_tien)

Kết quả:

Tổng số tiền đi chợ: 90000.0
Giải thích: 2 * 15000 = 30000 và 0.5 * 120000 = 60000, tổng cộng là 90000.

14. Ví dụ thực hành: Tính số tiền còn lại

Nếu ban đầu có 91.000đ và đi chợ hết 90.000đ, ta tính số tiền còn lại bằng phép trừ. 

so_tien_ban_dau = 91000

gia_rau = 15000
gia_thit = 120000

tong_so_tien = 2 * gia_rau + 0.5 * gia_thit
so_tien_con_lai = so_tien_ban_dau - tong_so_tien

print("Tổng số tiền đi chợ:", tong_so_tien)
print("Số tiền thừa sau khi đi chợ:", so_tien_con_lai)

Kết quả:

Tổng số tiền đi chợ: 90000.0
Số tiền thừa sau khi đi chợ: 1000.0

15. Lỗi thường gặp

Lỗi Nguyên nhân Cách sửa
Dùng x để nhân Python không hiểu 3 x 10 Dùng 3 * 10
Nhầm / và // / chia thật, // chia làm tròn xuống Chọn đúng toán tử theo yêu cầu
Nhầm % là phần trăm Trong Python, % là chia lấy dư Dùng 10 % 3 để lấy dư
Quên ngoặc khi cần ưu tiên x + y / 3 khác (x + y) / 3 Dùng dấu ngoặc rõ ràng
Nhập số từ bàn phím nhưng không ép kiểu input() trả về chuỗi Dùng int(input(...)) hoặc float(input(...))

16. Bài tập thực hành

Bài tập 1: Tính tổng tiền mua hàng

Nhập đơn giá và số lượng, sau đó tính tổng tiền.

don_gia = float(input("Nhập đơn giá: "))
so_luong = float(input("Nhập số lượng: "))

tong_tien = don_gia * so_luong
print("Tổng tiền:", tong_tien)

Bài tập 2: Tính chu vi và diện tích hình chữ nhật

Nhập chiều dài và chiều rộng, sau đó tính chu vi và diện tích.

chieu_dai = float(input("Nhập chiều dài: "))
chieu_rong = float(input("Nhập chiều rộng: "))

chu_vi = (chieu_dai + chieu_rong) * 2
dien_tich = chieu_dai * chieu_rong

print("Chu vi:", chu_vi)
print("Diện tích:", dien_tich)

Bài tập 3: Tách phần nguyên và phần dư

Nhập hai số nguyên n, d. In ra n // d và n % d.

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • + dùng để cộng.
  • - dùng để trừ.
  • * dùng để nhân.
  • / dùng để chia và luôn trả về float.
  • % dùng để chia lấy dư.
  • ** dùng để tính lũy thừa.
  • // dùng để chia làm tròn xuống.
  • +=, -=, *=, /= là các toán tử gán kết hợp.
  • Nên dùng ngoặc () để làm rõ thứ tự tính toán.
Mẹo học: hãy chạy thử các phép toán với cả số dương, số âm, số nguyên và số thực để hiểu rõ sự khác nhau giữa /, // và %.
