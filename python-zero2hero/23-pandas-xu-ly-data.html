23. Pandas — Làm việc và Xử lý Dữ liệu

Pandas là thư viện Python nền tảng cho xử lý dữ liệu dạng bảng. Bài này đi từ cài đặt, hai cấu trúc dữ liệu cốt lõi Series và DataFrame, đọc/ghi CSV/Excel, truy xuất với loc/iloc, lọc, chỉnh sửa, xử lý NaN, đến groupby, ghép bảng và làm sạch dữ liệu thực tế.

Giới thiệu

Trong thực tế, dữ liệu đến từ nhiều nguồn: file CSV, Excel, JSON, cơ sở dữ liệu SQL, API, log hệ thống, thiết bị mạng, IoT. Nếu dùng Python thuần với list, dict, vòng lặp for để xử lý vài trăm nghìn dòng, code sẽ rất dài và chậm.

Pandas giải quyết điều đó bằng cách:

  • Đọc dữ liệu vào Python từ nhiều định dạng.
  • Kiểm tra, làm sạch, lọc, sửa dữ liệu.
  • Tính toán thống kê, nhóm dữ liệu.
  • Ghép nhiều bảng.
  • Chuẩn bị dữ liệu cho trực quan hóa và Machine Learning.
Tư duy Pandas quan trọng hơn nhớ lệnh. Quy trình: đọc → kiểm tra → làm sạch → truy xuất/lọc → biến đổi → thống kê → xuất kết quả. Khi quen quy trình này, bạn tự tra cứu lệnh cụ thể rất nhanh.

Ví dụ nhanh so sánh Python thuần và Pandas

# Python thuần — lọc sinh viên điểm > 8
students = [
    {"name": "An", "class": "D24CQVT01", "score": 8.5},
    {"name": "Bình", "class": "D24CQVT01", "score": 7.0},
    {"name": "Chi", "class": "D24CQCN01", "score": 9.2},
]
high_score = [s for s in students if s["score"] > 8]

# Pandas — ngắn hơn, nhanh hơn, dễ mở rộng hơn
import pandas as pd

df = pd.read_csv("students.csv")
high_score = df[df["score"] > 8]
diem_tb_lop = df.groupby("class")["score"].mean()

6. Truy xuất dữ liệu: loc & iloc

Lấy cột

# Một cột → Series
names = df["name"]

# Nhiều cột → DataFrame
subset = df[["name", "score"]]

# Lưu ý: df["name"] khác df[["name"]]
# df["name"]   → Series
# df[["name"]] → DataFrame có 1 cột

loc — theo label

# Set index theo mã sinh viên
df = df.set_index("student_id")

# Lấy một dòng theo label
print(df.loc["SV001"])

# Lấy một ô cụ thể
print(df.loc["SV001", "name"])

# Lấy nhiều dòng
print(df.loc[["SV001", "SV003"]])

# Lấy nhiều dòng, nhiều cột
print(df.loc[["SV001", "SV003"], ["name", "score"]])

# Slice theo label (bao gồm cả điểm cuối — khác iloc)
print(df.loc["SV001":"SV003"])

iloc — theo vị trí số nguyên

# Lấy một dòng theo vị trí
print(df.iloc[0])         # dòng đầu tiên

# Lấy một ô cụ thể
print(df.iloc[0, 2])      # dòng 0, cột 2

# Nhiều dòng liên tiếp (không bao gồm điểm cuối — như Python slice)
print(df.iloc[0:3])

# Nhiều dòng, nhiều cột
print(df.iloc[0:3, 0:2])

# Các dòng/cột rời rạc
print(df.iloc[[0, 2], [1, 3]])
loc
Theo label/index name
loc["SV001", "name"]
Slice bao gồm điểm cuối
vs
iloc
Theo vị trí số nguyên
iloc[0, 2]
Slice không bao gồm điểm cuối
Nhầm loc với iloc là lỗi rất phổ biến. Quy tắc ghi nhớ: loc hỏi "hàng/cột tên gì?", iloc hỏi "hàng/cột đứng thứ mấy?". Nếu index là "SV001", viết df.iloc["SV001"] sẽ báo lỗi ngay.

at và iat — truy xuất ô đơn nhanh hơn

# at: theo label — nhanh hơn loc khi chỉ lấy/sửa một ô
print(df.at["SV001", "name"])
df.at["SV001", "score"] = 9.0

# iat: theo vị trí — nhanh hơn iloc khi chỉ lấy/sửa một ô
print(df.iat[0, 2])
df.iat[0, 2] = 9.0

7. Lọc dữ liệu

Lọc theo một điều kiện

df = pd.read_csv("students.csv")

# Sinh viên điểm >= 8
high_score = df[df["score"] >= 8]

# Sinh viên lớp D24CQVT01
vt_class = df[df["class"] == "D24CQVT01"]

# Sinh viên không phải lớp D24CQVT01 (toán tử ~)
other_class = df[~(df["class"] == "D24CQVT01")]

Lọc nhiều điều kiện — dùng &, |, ~

# AND: cả hai điều kiện đúng
result = df[(df["score"] >= 8) & (df["class"] == "D24CQVT01")]

# OR: ít nhất một điều kiện đúng
result = df[(df["score"] >= 9) | (df["class"] == "D24CQCN01")]

# Kết hợp nhiều
result = df[
    (df["score"] >= 8) &
    (df["age"] >= 18) &
    (df["class"] == "D24CQVT01")
]
Phải có ngoặc đơn bao quanh mỗi điều kiện. df[df["score"] >= 8 & df["age"] >= 18] → lỗi vì Python hiểu sai độ ưu tiên toán tử. Luôn viết (df["score"] >= 8) & (df["age"] >= 18). Đồng thời không dùng and/or của Python — phải dùng &/|.

Lọc nâng cao

# isin — thuộc tập hợp giá trị
result = df[df["class"].isin(["D24CQVT01", "D24CQVT02"])]

# between — trong khoảng giá trị
result = df[df["score"].between(7.0, 9.0)]

# str.contains — tìm chuỗi trong cột text
result = df[df["name"].str.contains("An")]

# query — cú pháp gần giống SQL
result = df.query("score >= 8 and class == 'D24CQVT01'")

# Lọc bằng loc + điều kiện
result = df.loc[df["score"] >= 8, ["name", "score"]]

Thêm cột mới

# Thêm cột tính từ cột khác
df["revenue"] = df["orders"] * df["unit_price"]

# Sửa một ô cụ thể
df.at["SV001", "score"] = 9.0
df.loc["SV001", "score"] = 9.0

# Sửa theo điều kiện — PHẢI dùng loc
df.loc[df["score"] >= 8, "rank"] = "Giỏi"

SettingWithCopyWarning — cạm bẫy phổ biến. Không bao giờ sửa dữ liệu qua một bản lọc tạm: df_high = df[df["score"] >= 8]; df_high["rank"] = "Giỏi" — Pandas có thể không sửa vào df gốc. Dùng df.loc[điều_kiện, "rank"] = "Giỏi" hoặc df_high = df[...].copy() rồi sửa trên bản copy.

9. Xử lý dữ liệu thiếu NaN

Dữ liệu thực tế luôn có giá trị thiếu. Pandas dùng NaN (Not a Number) từ NumPy hoặc pd.NA để biểu diễn giá trị thiếu. Xử lý sai sẽ dẫn đến sai kết quả thống kê.

Phát hiện giá trị thiếu

# Đếm số NaN mỗi cột
print(df.isna().sum())

# Tỷ lệ % NaN
print(df.isna().mean() * 100)

# Kiểm tra từng ô
print(df.isna())            # DataFrame True/False

# Xem các dòng có NaN
print(df[df.isna().any(axis=1)])

# Dòng có NaN ở cột cụ thể
print(df[df["score"].isna()])

Xóa hàng/cột có NaN

# Xóa mọi hàng có ít nhất một NaN
df_clean = df.dropna()

# Xóa hàng chỉ khi có NaN ở cột cụ thể
df_clean = df.dropna(subset=["score", "name"])

# Xóa cột có hơn 50% NaN
threshold = len(df) * 0.5
df_clean = df.dropna(thresh=threshold, axis=1)

Điền giá trị NaN

# Điền NaN bằng giá trị cố định
df["city"] = df["city"].fillna("Unknown")
df["score"] = df["score"].fillna(0)

# Điền NaN bằng giá trị thống kê
mean_score = df["score"].mean()
df["score"] = df["score"].fillna(mean_score)

median_age = df["age"].median()
df["age"] = df["age"].fillna(median_age)

# Điền theo giá trị trước đó (forward fill) — dữ liệu chuỗi thời gian
df["sensor"] = df["sensor"].fillna(method="ffill")

# Điền theo giá trị sau đó (backward fill)
df["sensor"] = df["sensor"].fillna(method="bfill")

# Nội suy tuyến tính
df["temperature"] = df["temperature"].interpolate()
Không điền NaN bừa. Phải hiểu ý nghĩa dữ liệu thiếu trước. Điền mean phù hợp với dữ liệu phân phối chuẩn. Điền median khi có outlier. Điền forward fill với dữ liệu sensor. Điền "Unknown" với cột text. Xóa hàng khi số NaN ít và dữ liệu không quan trọng.

10. GroupBy & Thống kê

groupby cơ bản

df = pd.read_csv("students.csv")

# Điểm trung bình theo lớp
print(df.groupby("class")["score"].mean())

# Số sinh viên theo lớp
print(df.groupby("class")["name"].count())

# Điểm cao nhất theo lớp
print(df.groupby("class")["score"].max())

# Tổng hợp nhiều phép tính
summary = df.groupby("class").agg(
    so_sv=("name", "count"),
    diem_tb=("score", "mean"),
    diem_max=("score", "max"),
    diem_min=("score", "min"),
)
print(summary)

groupby nhiều cột

# Nhóm theo nhiều cột
result = df.groupby(["class", "gender"])["score"].mean()

# Reset index để ra DataFrame phẳng
result = df.groupby("class", as_index=False)["score"].mean()
print(result)
#        class     score
# 0  D24CQCN01  7.900000
# 1  D24CQVT01  8.200000

Thống kê nhanh trên một cột

col = df["score"]

print(col.sum())       # tổng
print(col.mean())      # trung bình
print(col.median())    # trung vị
print(col.std())       # độ lệch chuẩn
print(col.min())       # nhỏ nhất
print(col.max())       # lớn nhất
print(col.count())     # số giá trị không NaN
print(col.quantile(0.75))  # phân vị Q3

pivot_table

import pandas as pd
import numpy as np

11. Ghép bảng

concat — ghép dọc hoặc ngang

df1 = pd.DataFrame({"name": ["An", "Bình"], "score": [8.5, 7.0]})
df2 = pd.DataFrame({"name": ["Chi", "Dũng"], "score": [9.2, 6.8]})

# Ghép dọc (stack rows)
df_all = pd.concat([df1, df2], ignore_index=True)

# Ghép ngang (stack columns)
df_wide = pd.concat([df1, df2], axis=1)

merge — join như SQL

students = pd.DataFrame({
    "student_id": ["SV001", "SV002", "SV003"],
    "name": ["An", "Bình", "Chi"],
    "class_id": [1, 1, 2]
})

classes = pd.DataFrame({
    "class_id": [1, 2],
    "class_name": ["D24CQVT01", "D24CQCN01"]
})

# INNER JOIN — chỉ giữ hàng có key khớp cả hai bảng
result = pd.merge(students, classes, on="class_id")

# LEFT JOIN — giữ tất cả hàng bên trái
result = pd.merge(students, classes, on="class_id", how="left")

# RIGHT JOIN
result = pd.merge(students, classes, on="class_id", how="right")

# OUTER JOIN
result = pd.merge(students, classes, on="class_id", how="outer")

# Merge với tên cột khác nhau
result = pd.merge(students, classes, left_on="class_id", right_on="id")
howHành viTương đương SQL
"inner"Chỉ giữ key khớp cả hai bảngINNER JOIN
"left"Giữ tất cả hàng bảng tráiLEFT JOIN
"right"Giữ tất cả hàng bảng phảiRIGHT JOIN
"outer"Giữ tất cả hàng cả hai bảngFULL OUTER JOIN

14. Bài tập thực hành

Bài 1 — Tạo DataFrame sinh viên

Tạo DataFrame có ít nhất 6 sinh viên với các cột student_id, name, class, score, age. In ra 3 dòng đầu, tên cột, kiểu dữ liệu và thống kê tổng quát.

Bài 2 — Đọc, kiểm tra & ghi file

Tạo file students.csv từ bài 1. Đọc lại bằng pd.read_csv(). Chạy head, info, describe, isna().sum(). Xuất ra students_output.csv không ghi index.

Bài 3 — Lọc dữ liệu

Từ DataFrame sinh viên: lọc sinh viên có score >= 8, lọc theo lớp cụ thể, lọc kết hợp nhiều điều kiện. Dùng cả isin(), between() và query().

Bài 4 — loc và iloc

Set student_id làm index. Dùng iloc lấy dòng đầu tiên và ô [0, 2]. Dùng loc lấy sinh viên theo mã và nhiều cột. So sánh kết quả loc vs iloc.

Bài 5 — Xử lý NaN

Thêm một số giá trị NaN vào cột score và age. Đếm NaN, xóa dòng thiếu, điền điểm thiếu bằng trung bình, điền age thiếu bằng median. So sánh kết quả trước và sau khi xử lý bằng describe().

Bài 6 — GroupBy

Từ dữ liệu sinh viên: tính điểm trung bình theo lớp, đếm sinh viên theo lớp, tìm điểm cao nhất theo lớp bằng agg() một lần. Xuất kết quả ra Excel hai sheet.

Bài 7 — Làm sạch text

Tạo cột name có khoảng trắng dư và chữ hoa lộn xộn. Xóa khoảng trắng, chuẩn hóa title case. Tách họ và tên thành hai cột riêng. Tìm sinh viên có tên chứa "An".

Bài 8 — Mini Report

Từ file students.csv, tạo báo cáo: tổng số sinh viên, điểm trung bình toàn trường, điểm TB theo lớp, danh sách sinh viên giỏi (score >= 8). Xuất ra: report_by_class.xlsx và good_students.csv. 

import pandas as pd

df = pd.read_csv("students.csv")

# Thống kê tổng
print(f"Tổng sinh viên: {len(df)}")
print(f"Điểm TB toàn trường: {df['score'].mean():.2f}")

# Điểm TB theo lớp
by_class = df.groupby("class", as_index=False).agg(
    so_sv=("name", "count"),
    diem_tb=("score", "mean"),
    diem_max=("score", "max"),
)

# Sinh viên giỏi
good = df[df["score"] >= 8][["student_id", "name", "class", "score"]]
good = good.sort_values("score", ascending=False)

# Xuất file
with pd.ExcelWriter("report_by_class.xlsx") as writer:
    by_class.to_excel(writer, sheet_name="ByClass", index=False)
    df.describe().to_excel(writer, sheet_name="Summary")

good.to_csv("good_students.csv", index=False, encoding="utf-8-sig")

print("Xuất file hoàn tất.")

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Series là dữ liệu một chiều có nhãn — mỗi cột trong DataFrame là một Series.
  • DataFrame là bảng dữ liệu hai chiều — đơn vị làm việc chính của Pandas.
  • Đọc/ghi: pd.read_csv(), pd.read_excel(), to_csv(), to_excel(); nhớ encoding và index=False.
  • Kiểm tra trước khi xử lý: luôn chạy head, info, describe, isna().sum().
  • loc theo label, iloc theo vị trí số — không được nhầm hai cái này.
  • Lọc nhiều điều kiện: luôn dùng &, |, ~ với ngoặc đơn bao mỗi điều kiện.
  • Sửa dữ liệu theo điều kiện: dùng df.loc[điều_kiện, "cột"] = ... — không sửa qua bản lọc tạm.
  • NaN: phân biệt khi nào dùng dropna, fillna(mean), fillna(median), fillna("Unknown"), ffill/bfill.
  • groupby + agg: nhóm dữ liệu và tổng hợp nhiều phép tính một lần.
  • merge: join bảng như SQL với how="inner"/"left"/"right"/"outer".
  • str accessor: xử lý text với .str.strip(), .str.lower(), .str.contains(), .str.split().
  • dt accessor: trích xuất year/month/day và làm việc với chuỗi thời gian.
Lộ trình tiếp theo: Sau Pandas, bài 24 sẽ đi vào SQL/SQLite — nơi bạn thấy groupby tương đương GROUP BY, merge tương đương JOIN. Kết hợp Pandas + SQL là bộ kỹ năng xử lý dữ liệu cốt lõi cho mọi hướng: Data Engineering, Machine Learning, Network Automation, hay DevSecOps.
