04. Các phép toán so sánh và logic trong Python

Tài liệu học Python cơ bản về comparison operators và logical operators: so sánh bằng, không bằng, lớn hơn, nhỏ hơn, kết hợp điều kiện với and / or / not và áp dụng vào các bài toán thực tế.

Giới thiệu

Phép toán so sánh dùng để so sánh hai giá trị với nhau. Kết quả luôn là kiểu boolean: True hoặc False.

Phép toán logic dùng để kết hợp các điều kiện (conditional statements) lại với nhau. Một conditional statement là bất kỳ biểu thức nào chỉ nhận giá trị True hoặc False — ví dụ: một biến boolean, một phép so sánh, hoặc kết quả gọi hàm.

Mục tiêu bài học: nắm vững 6 phép toán so sánh, 3 phép toán logic (and, or, not), hiểu bảng chân trị và biết kết hợp chúng để giải các bài toán kiểm tra điều kiện thực tế.

1. Bảng phép toán so sánh

Phép toán Ký hiệu Ví dụ Kết quả
Bằng nhau == 20 == 20 True
Không bằng != 20 != 10 True
Lớn hơn > 20 > 10 True
Nhỏ hơn < 15 < 10 False
Lớn hơn hoặc bằng >= 10 >= 10 True
Nhỏ hơn hoặc bằng <= 15 <= 15 True
Dễ nhầm: = là toán tử gán giá trị (x = 5), còn == mới là toán tử so sánh bằng (x == 5). Nhầm lẫn hai ký hiệu này là lỗi rất phổ biến.

2. Bảng phép toán logic

Toán tử Cú pháp Kết quả True khi nào?
and stmt1 and stmt2 Cả hai điều kiện đều True
or stmt1 or stmt2 Ít nhất một điều kiện là True
not not stmt Điều kiện gốc là False (phủ định)

3. Bảng chân trị — and

Toán tử and chỉ trả về True khi tất cả các vế đều đúng.

stmt1 stmt2 stmt1 and stmt2
True True True
True False False
False True False
False False False

4. Bảng chân trị — or

Toán tử or trả về True khi ít nhất một vế đúng. Chỉ trả về False khi cả hai vế đều sai.

stmt1 stmt2 stmt1 or stmt2
True True True
True False True
False True True
False False False

5. Bảng chân trị — not

Toán tử not đảo ngược giá trị boolean của điều kiện.

stmt not stmt
True False
False True

6. Ví dụ: Phép toán so sánh

Phép so sánh áp dụng được cho cả số nguyên, số thực lẫn chuỗi.

So sánh số

# Kiểm tra bằng nhau
print(20 == 20)   # True
print(20 == 10)   # False

# Kiểm tra không bằng
print(20 != 10)   # True

# Lớn hơn / nhỏ hơn
print(20 > 10)    # True
print(10 >= 10)   # True
print(15 < 10)    # False
print(15 <= 15)   # True

Kết quả:

True
False
True
True
True
False
True

So sánh chuỗi

Python so sánh chuỗi theo thứ tự từ điển (lexicographic). Toán tử == kiểm tra hai chuỗi có giống nhau từng ký tự không. 

print('anh' == 'anh')   # True
print('anh' == 'em')    # False
print('anh' != 'em')    # True

Kết quả:

True
False
True

7. Ví dụ: Kiểm tra số trong khoảng (and)

Kiểm tra xem một số có nằm trong đoạn [10, 20] không — tức là phải đồng thời lớn hơn hoặc bằng 10 và nhỏ hơn hoặc bằng 20. 

number = 15
in_range = (number >= 10) and (number <= 20)
print("Số nằm trong khoảng [10, 20]:", in_range)

Kết quả:

Số nằm trong khoảng [10, 20]: True

Cú pháp gọn hơn — chained comparison

Python hỗ trợ viết chuỗi so sánh liên tiếp rất tự nhiên, giống ký hiệu toán học:

number = 15
in_range = 10 <= number <= 20
print("Số nằm trong khoảng [10, 20]:", in_range)

Kết quả:

Số nằm trong khoảng [10, 20]: True
Ghi nhớ: cú pháp 10 <= number <= 20 hoàn toàn hợp lệ trong Python và tương đương với (10 <= number) and (number <= 20).

8. Ví dụ: Kiểm tra số ngoài khoảng (or)

Kiểm tra xem một số có nằm ngoài đoạn [10, 20] không — tức là nhỏ hơn 10 hoặc lớn hơn 20. 

number = 30
out_of_range = (number < 10) or (number > 20)
print("Số nằm ngoài khoảng [10, 20]:", out_of_range)

Kết quả:

Số nằm ngoài khoảng [10, 20]: True

Giải thích với number = 30:

  • 30 < 10 → False
  • 30 > 20 → True
  • False or True → True ✓ (chỉ cần một vế đúng)

9. Ví dụ: Phủ định (not)

Toán tử not đặt trước một điều kiện để lấy giá trị ngược lại. 

b = False
print(not b)          # True — phủ định False

print(not (6 > 5))    # False — 6 > 5 là True, not True = False

Kết quả:

True
False
Dùng khi nào? not hữu ích khi bạn đã có một biến boolean và muốn kiểm tra trường hợp ngược lại, ví dụ: if not is_logged_in: thay vì if is_logged_in == False:.

10. Kết hợp nhiều điều kiện

Có thể nối nhiều điều kiện trong một biểu thức bằng cách dùng nhiều and / or liên tiếp: 

age = 30
has_id = True
gender = 'Nam'

result = (age >= 18) and has_id and (gender == 'Nam')
print("Thỏa mãn tất cả điều kiện:", result)

Kết quả:

Thỏa mãn tất cả điều kiện: True

Khi cần nối nhiều điều kiện mà có cả and lẫn or, thứ tự ưu tiên mặc định là: not > and > or. Để rõ ràng hơn, hãy dùng dấu ngoặc.

Mức ưu tiên Toán tử
Cao nhất not
Trung bình and
Thấp nhất or

11. Dùng dấu ngoặc để làm rõ thứ tự ưu tiên

Khi kết hợp nhiều phép toán, luôn nên dùng dấu ngoặc để tránh nhầm lẫn và dễ đọc hơn. Đây cũng là best practice được khuyến nghị. 

# Không có ngoặc — dễ nhầm thứ tự thực thi
result = a or b and c          # b and c được tính trước

# Có ngoặc — rõ ràng hơn
result = a or (b and c)        # tường minh ý định
result = (a or b) and c        # thay đổi kết quả hoàn toàn
Lời khuyên: khi chưa chắc thứ tự ưu tiên, hãy luôn dùng dấu ngoặc để làm rõ điều kiện nào được tính trước. Code rõ ràng quan trọng hơn code ngắn.

12. Ví dụ thực hành: Kiểm tra vào club

Yêu cầu: người dùng có được vào club không? Điều kiện: tuổi ≥ 18 và có ID. 

age = 20
has_id = True

enter_club = (age >= 18) and has_id
print("User có được vào club:", enter_club)

Kết quả:

User có được vào club: True

Thêm điều kiện thứ ba

Mở rộng bài toán: club chỉ dành cho nam giới.

age = 30
has_id = True
gender = 'Nam'

enter_club = (age >= 18) and has_id and (gender == 'Nam')
print("User có được vào club:", enter_club)

Kết quả:

User có được vào club: True

Kiểm tra các trường hợp

age has_id gender Kết quả Lý do
20 True 'Nam' True Tất cả điều kiện thỏa mãn
15 True 'Nam' False Tuổi không đủ 18
30 False 'Nam' False Không có ID
30 True 'Nữ' False Không phải nam giới

13. Ví dụ thực hành: Kiểm tra vào rạp chiếu phim

Yêu cầu: người dùng có được vào rạp chiếu phim không? Điều kiện:

  • Phải có vé — điều kiện tiên quyết.
  • Tuổi ≥ 18, hoặc nhỏ hơn 18 nhưng có bố mẹ đi kèm.
age = 15
has_ticket = True
has_parent = False

# Điều kiện tuổi tác: đủ 18 HOẶC có bố mẹ đi kèm
age_condition = (age >= 18) or has_parent

# Điều kiện đầy đủ: phải có vé VÀ thỏa mãn điều kiện tuổi
can_enter = has_ticket and age_condition

print("Có thể vào rạp chiếu phim:", can_enter)

Kết quả với age=15, has_ticket=True, has_parent=False:

Có thể vào rạp chiếu phim: False

Giải thích từng bước:

  • age >= 18 → 15 >= 18 → False
  • has_parent → False
  • age_condition = False or False → False
  • can_enter = True and False → False

Kiểm tra trường hợp có bố mẹ đi kèm

age = 15
has_ticket = True
has_parent = True

age_condition = (age >= 18) or has_parent   # False or True = True
can_enter = has_ticket and age_condition    # True and True  = True

print("Có thể vào rạp chiếu phim:", can_enter)

Kết quả:

Có thể vào rạp chiếu phim: True

Kiểm tra trường hợp không có vé

age = 40
has_ticket = False
has_parent = True

age_condition = (age >= 18) or has_parent   # True or True = True
can_enter = has_ticket and age_condition    # False and True = False

print("Có thể vào rạp chiếu phim:", can_enter)

Kết quả:

Có thể vào rạp chiếu phim: False
Bài học: trong biểu thức and, chỉ cần một vế là False thì toàn bộ kết quả là False. Do đó điều kiện tiên quyết như "phải có vé" rất phù hợp đặt vào vế and đầu tiên.

15. Bài tập thực hành

Bài tập 1: Kiểm tra số chẵn/lẻ

Nhập một số nguyên. Kiểm tra xem số đó có phải số chẵn không và in ra kết quả True / False.

n = int(input("Nhập một số nguyên: "))
is_even = (n % 2 == 0)
print("Là số chẵn:", is_even)

Bài tập 2: Kiểm tra tam giác hợp lệ

Nhập ba cạnh a, b, c. Kiểm tra xem chúng có thể tạo thành một tam giác không (mỗi cạnh phải nhỏ hơn tổng hai cạnh còn lại).

a = float(input("Nhập cạnh a: "))
b = float(input("Nhập cạnh b: "))
c = float(input("Nhập cạnh c: "))

is_valid = (a + b > c) and (a + c > b) and (b + c > a)
print("Là tam giác hợp lệ:", is_valid)

Bài tập 3: Phân loại tuổi

Nhập tuổi. In ra True nếu người đó là thiếu niên (từ 13 đến 17 tuổi), ngược lại in ra False.

Bài tập 4: Kiểm tra đăng nhập đơn giản

Cho trước correct_username = "admin" và correct_password = "1234". Nhập username và password từ bàn phím. Kiểm tra xem cả hai có khớp không và in ra kết quả.

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Có 6 phép toán so sánh: ==, !=, >, <, >=, <= — kết quả luôn là True hoặc False.
  • == là so sánh bằng; = là gán giá trị — không được nhầm lẫn.
  • and: tất cả vế phải đúng → kết quả mới là True.
  • or: chỉ cần một vế đúng → kết quả là True.
  • not: đảo ngược giá trị boolean.
  • Thứ tự ưu tiên: not > and > or. Luôn dùng () khi kết hợp để tránh nhầm.
  • Python hỗ trợ chained comparison: 10 <= x <= 20 thay cho (x >= 10) and (x <= 20).
Mẹo học: phép toán so sánh và logic là nền tảng của if/else, while và lọc dữ liệu — hãy thử gán kết quả vào biến rồi in ra để kiểm tra từng trường hợp trước khi tích hợp vào cấu trúc điều kiện lớn hơn.
