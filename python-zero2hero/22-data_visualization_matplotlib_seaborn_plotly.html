22. Data Visualization — Matplotlib, Seaborn & Plotly

Trực quan hóa dữ liệu là cách biến những con số khô khan thành hình ảnh dễ hiểu — giúp phát hiện xu hướng, phân bố, tương quan và bất thường. Bài này đi từ Matplotlib (nền tảng, tùy chỉnh sâu), qua Seaborn (thống kê nhanh, đẹp) đến Plotly (tương tác, dashboard).

Giới thiệu

Data Visualization là quá trình biểu diễn dữ liệu dưới dạng hình ảnh — biểu đồ, đồ thị, heatmap, dashboard — để con người dễ hiểu hơn so với việc đọc bảng số liệu dài.

Ví dụ thực tế:

  • Doanh thu theo tháng, theo khu vực
  • Mối quan hệ giữa chi phí quảng cáo và doanh thu
  • Phân bố điểm thi của sinh viên
  • Bandwidth, latency, top IP trong network monitoring
  • Heatmap lỗi theo thiết bị và thời gian
Vì sao quan trọng? Biểu đồ giúp hiểu dữ liệu trước khi xử lý, phát hiện outlier/dữ liệu thiếu, trình bày kết quả, theo dõi hệ thống realtime và hỗ trợ ra quyết định.

3. Dataset mẫu dùng xuyên suốt bài

Tạo một dataset giả lập về doanh thu bán hàng — sẽ dùng lại trong toàn bộ Matplotlib, Seaborn và Plotly để dễ so sánh. 

import numpy as np
import pandas as pd

np.random.seed(42)

dates = pd.date_range("2026-01-01", periods=120, freq="D")
regions = ["North", "South", "East", "West"]
products = ["Laptop", "Mouse", "Keyboard", "Monitor"]

df = pd.DataFrame({
    "date": np.random.choice(dates, 500),
    "region": np.random.choice(regions, 500),
    "product": np.random.choice(products, 500),
    "orders": np.random.randint(1, 20, 500),
    "unit_price": np.random.randint(10, 1200, 500),
})

df["revenue"] = df["orders"] * df["unit_price"]
df["profit"] = df["revenue"] * np.random.uniform(0.08, 0.35, 500)
df["month"] = df["date"].dt.to_period("M").astype(str)

print(df.head())

import matplotlib.pyplot as plt

Nhầm Axes và Axis. Axes là vùng vẽ biểu đồ (một ax), còn Axis là trục X hoặc Y. fig, ax = plt.subplots() → ax là một Axes, không phải một trục đơn.

Pyplot API vs Object-Oriented API

# Cách 1: Pyplot API — nhanh, đơn giản
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [2, 4, 6])
plt.title("Line Chart")
plt.xlabel("X")
plt.ylabel("Y")
plt.show()

# Cách 2: Object-Oriented API — khuyến nghị
fig, ax = plt.subplots()

ax.plot([1, 2, 3], [2, 4, 6])
ax.set_title("Line Chart")
ax.set_xlabel("X")
ax.set_ylabel("Y")

plt.show()
Nên ưu tiên Object-Oriented API vì code rõ ràng hơn, dễ quản lý nhiều biểu đồ, dễ tùy chỉnh chi tiết và phù hợp khi viết project lớn.

Scatter, Line, Histplot, KDE

import seaborn as sns
import matplotlib.pyplot as plt

sns.set_theme(style="whitegrid")

# Scatter — hue tự tô màu theo nhóm, size tự đổi kích thước điểm
fig, ax = plt.subplots(figsize=(8, 4))
sns.scatterplot(
    data=df, x="orders", y="revenue",
    hue="region", size="profit", ax=ax
)
ax.set_title("Mối quan hệ giữa số đơn và doanh thu")
plt.show()

# Line — theo nhóm
monthly = (
    df.groupby(["month", "region"], as_index=False)
      .agg(revenue=("revenue", "sum"))
)

fig, ax = plt.subplots(figsize=(10, 4))
sns.lineplot(data=monthly, x="month", y="revenue", hue="region", marker="o", ax=ax)
ax.set_title("Doanh thu theo tháng và khu vực")
plt.xticks(rotation=45)
plt.show()

# Histplot + KDE — phân bố dữ liệu
fig, ax = plt.subplots(figsize=(8, 4))
sns.histplot(data=df, x="revenue", bins=30, kde=True, ax=ax)
ax.set_title("Phân bố doanh thu")
plt.show()
Điểm mạnh của Seaborn: hue="region" tự tô màu theo nhóm, size="profit" tự đổi kích thước điểm, và Seaborn tự tạo legend — không cần code thêm.

# Axes-level — vẽ vào một ax cụ thể (cần ax=ax)
fig, ax = plt.subplots()
sns.scatterplot(data=df, x="orders", y="revenue", ax=ax)

# Figure-level — tự tạo Figure riêng, hỗ trợ col/row để chia nhỏ
sns.relplot(
    data=df, x="orders", y="revenue",
    hue="product", col="region", kind="scatter", col_wrap=2, height=4
)

8. Plotly — biểu đồ tương tác và dashboard

Plotly tạo biểu đồ cho phép người xem hover xem giá trị, zoom, pan, bật/tắt nhóm trong legend và xuất ra HTML để mở trong trình duyệt hoặc nhúng vào web.

Cách dùngĐặc điểm
Plotly Express (px)Nhanh, ít code, phù hợp đa số trường hợp
Graph Objects (go)Tùy chỉnh sâu, biểu đồ phức tạp, secondary axis

Line, Scatter, Bar, Histogram, Box

import plotly.express as px

# Line chart
monthly = (
    df.groupby(["month", "region"], as_index=False)
      .agg(revenue=("revenue", "sum"))
)

fig = px.line(
    monthly, x="month", y="revenue", color="region",
    markers=True, title="Doanh thu theo tháng và khu vực"
)
fig.show()

# Scatter — size, hover_data
fig = px.scatter(
    df, x="orders", y="revenue", color="region", size="profit",
    hover_data=["product", "unit_price"], title="Orders vs Revenue"
)
fig.show()

# Bar — barmode group/stack/relative
summary = (
    df.groupby(["product", "region"], as_index=False)
      .agg(total_revenue=("revenue", "sum"))
)

fig = px.bar(
    summary, x="product", y="total_revenue", color="region",
    barmode="group", title="Tổng doanh thu theo sản phẩm và khu vực"
)
fig.show()

# Histogram với marginal box plot
fig = px.histogram(
    df, x="revenue", color="region", nbins=30,
    marginal="box", title="Phân bố doanh thu theo khu vực"
)
fig.show()

# Box plot
fig = px.box(
    df, x="region", y="revenue", color="region",
    points="outliers", title="Box Plot doanh thu theo khu vực"
)
fig.show()

9. Quy trình chọn biểu đồ phù hợp

Câu hỏiLoại biểu đồVí dụ
Dữ liệu thay đổi theo thời gian? Line chart, Area chart, Time series Doanh thu theo tháng, CPU usage, network traffic
Muốn so sánh các nhóm? Bar chart, Horizontal bar, Grouped bar Doanh thu theo sản phẩm, top IP nhiều packet nhất
Muốn xem phân bố? Histogram, KDE, Box plot, Violin plot Phân bố điểm thi, latency, response time
Muốn xem quan hệ giữa 2 biến? Scatter plot, Regression plot, Bubble chart Giờ học & điểm thi, packet size & latency
Muốn xem tương quan nhiều biến? Heatmap, Pair plot, Correlation matrix Tương quan revenue/profit/orders/unit_price
Cần tương tác (hover/zoom)? Plotly Dashboard doanh thu, network monitoring

Checklist biểu đồ chuyên nghiệp

  • Có tiêu đề rõ ràng chưa?
  • Trục X, trục Y có nhãn và đơn vị rõ chưa?
  • Legend có cần thiết không, có dễ đọc không?
  • Màu sắc có dễ phân biệt, không quá nhiều màu/category?
  • Có cần sắp xếp dữ liệu hoặc format số lớn trước khi vẽ?
  • Có cần xoay nhãn trục X cho khỏi chồng?
  • Đã chọn đúng định dạng xuất (PNG/SVG/PDF/HTML)?
  • Biểu đồ có truyền tải đúng insight chính không?

11. Bài tập thực hành

Bài 1 — Line Plot cơ bản

Tạo mảng x = 0..10, vẽ 3 đường: y = x, y = x^2, y = 2x + 1. Yêu cầu: có title, label trục X/Y, legend và grid.

Bài 2 — Scatter Plot

Tạo 100 điểm ngẫu nhiên x, y. Vẽ scatter, thay đổi marker, thay đổi alpha, thêm colorbar.

Bài 3 — Bar Chart

Tạo dữ liệu doanh thu 5 sản phẩm. Vẽ bar chart, sắp xếp theo doanh thu giảm dần, thêm giá trị trên đầu mỗi cột (dùng ax.text() hoặc ax.bar_label()).

Bài 4 — Histogram + KDE

Tạo 1000 điểm theo phân phối normal. Vẽ histogram với 20 bins bằng Matplotlib, vẽ thêm KDE bằng Seaborn (sns.histplot(..., kde=True)). Nhận xét phân bố.

Bài 5 — Box Plot so sánh nhóm

Dùng dataset giả lập điểm 3 lớp. Vẽ box plot, so sánh median, tìm lớp có độ phân tán lớn nhất.

Bài 6 — Heatmap tương quan

Tạo DataFrame gồm 4 cột số (ví dụ orders, unit_price, revenue, profit). Tính correlation, vẽ heatmap, hiển thị số trong từng ô với annot=True.

Bài 7 — Seaborn hue + size

Dùng dataset bán hàng. Vẽ scatter orders vs revenue, phân màu theo region, kích thước theo profit.

Bài 8 — Seaborn Facet

Vẽ orders vs revenue bằng sns.relplot, chia biểu đồ theo từng region (col="region"), mỗi biểu đồ con tô màu theo product (hue="product").

Bài 9 — Plotly Interactive Chart

Vẽ line chart doanh thu theo tháng bằng px.line, phân màu theo region, có hover_data, xuất ra file chart.html bằng fig.write_html().

Bài 10 — Mini Dashboard

Tạo 3 biểu đồ: tổng doanh thu theo tháng, doanh thu theo sản phẩm, phân bố doanh thu theo khu vực. Làm một bản bằng Matplotlib/Seaborn (subplot 2×2 hoặc nhiều) và một bản tương tác bằng Plotly (xuất HTML). 

import matplotlib.pyplot as plt
import seaborn as sns

sns.set_theme(style="whitegrid")

monthly = df.groupby("month", as_index=False).agg(total_revenue=("revenue", "sum"))
product_summary = (
    df.groupby("product", as_index=False)
      .agg(total_revenue=("revenue", "sum"))
      .sort_values("total_revenue", ascending=False)
)
corr = df[["orders", "unit_price", "revenue", "profit"]].corr()

fig, axes = plt.subplots(2, 2, figsize=(14, 10))

sns.lineplot(data=monthly, x="month", y="total_revenue", marker="o", ax=axes[0, 0])
axes[0, 0].set_title("Tổng doanh thu theo tháng")
axes[0, 0].tick_params(axis="x", rotation=45)

sns.barplot(data=product_summary, x="product", y="total_revenue", ax=axes[0, 1])
axes[0, 1].set_title("Tổng doanh thu theo sản phẩm")

sns.histplot(data=df, x="revenue", bins=30, kde=True, ax=axes[1, 0])
axes[1, 0].set_title("Phân bố doanh thu")

sns.heatmap(corr, annot=True, cmap="coolwarm", fmt=".2f", ax=axes[1, 1])
axes[1, 1].set_title("Correlation Heatmap")

fig.suptitle("Báo cáo bán hàng", fontsize=18)
plt.tight_layout()
fig.savefig("sales_report.png", dpi=300, bbox_inches="tight")
plt.show()

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Matplotlib là nền tảng — hiểu Figure, Axes, Axis; ưu tiên Object-Oriented API (fig, ax = plt.subplots()).
  • Các loại biểu đồ cốt lõi: line, scatter, bar/barh, grouped bar, histogram, box plot, pie chart, subplots, time series.
  • Tùy chỉnh: title, label, legend, grid, style; luôn fig.savefig() trước plt.show().
  • Seaborn nhanh và đẹp cho DataFrame — hue tự phân nhóm; nhớ barplot mặc định lấy trung bình, cần groupby nếu muốn tổng.
  • Heatmap, pairplot, regplot giúp khám phá tương quan và xu hướng giữa nhiều biến.
  • Plotly cho biểu đồ tương tác — px nhanh, go tùy chỉnh sâu; xuất HTML bằng fig.write_html().
  • Chọn biểu đồ theo câu hỏi phân tích: thời gian → line; so sánh nhóm → bar; phân bố → histogram/box; quan hệ 2 biến → scatter; tương quan nhiều biến → heatmap/pairplot; cần tương tác → Plotly.
  • Luôn kiểm tra checklist: title, label, legend, màu sắc, định dạng xuất, và insight chính trước khi coi biểu đồ là hoàn thành.
Lộ trình tiếp theo: Matplotlib để hiểu cấu trúc → Seaborn để phân tích nhanh → Plotly để làm dashboard tương tác. Bài 23 (Pandas) và bài 24 (SQL/SQLite) sẽ cung cấp dữ liệu thực tế để áp dụng các kỹ thuật trực quan hóa này.
