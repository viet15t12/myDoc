08. Kiểu dữ liệu Set trong Python

Tài liệu học Python cơ bản về kiểu dữ liệu set: tạo set, thêm / xóa phần tử, duyệt qua set, và các phép toán tập hợp: giao, hợp, hiệu, hiệu đối xứng — bằng cả phương thức lẫn toán tử ký hiệu.

Giới thiệu

Set là một trong bốn kiểu collection của Python. Set có hai đặc điểm cốt lõi:

  • Các phần tử không được trùng lặp — set tự động loại bỏ giá trị trùng.
  • Không có thứ tự cố định — thứ tự phần tử khi duyệt có thể thay đổi mỗi lần chạy.
So sánh nhanh 4 kiểu collection phổ biến trong Python
Kiểu dữ liệu Ký hiệu Thay đổi được? Có thứ tự? Cho phép trùng lặp?
listDanh sách linh hoạt [...] Có Có Có
tupleDanh sách cố định (...) Không Có Có
setTập hợp giá trị duy nhất {...} Có Không Không
dictCặp khóa – giá trị {k: v} Có Có Không trùng key

Set hữu dụng khi bạn cần lưu danh sách giá trị duy nhất và thực hiện các phép toán tập hợp (giao, hợp, hiệu) một cách nhanh chóng.

1. Tạo set

Dùng cặp dấu ngoặc nhọn {} với ít nhất một phần tử, hoặc hàm set():

# Cách 1: dùng dấu ngoặc nhọn
my_set = {"a", "b", "c"}
print(my_set)        # {'a', 'b', 'c'}  (thứ tự có thể khác)
print(type(my_set))  # <class 'set'>

# Cách 2: chuyển từ list sang set
my_set2 = set([1, 2, 3, 4])
print(my_set2)       # {1, 2, 3, 4}
Phần tử của set phải là immutable (int, float, string, tuple). Không thể cho list hay dict vào set.

2. Set rỗng

Để tạo set rỗng, bạn bắt buộc phải dùng hàm set():

# ĐÚNG: set rỗng
empty_set = set()
print(type(empty_set))   # <class 'set'>

# SAI: đây là dictionary rỗng, không phải set!
not_a_set = {}
print(type(not_a_set))   # <class 'dict'>
Bẫy hay gặp: {} tạo ra dictionary rỗng, không phải set rỗng. Luôn dùng set() khi muốn có set không chứa phần tử.

3. Không cho phép trùng lặp

Khi khởi tạo hoặc thêm phần tử đã tồn tại, set tự động bỏ qua — không báo lỗi, không thêm bản sao. 

my_set = {"a", "a", "b"}
print(my_set)   # {'a', 'b'}  — chỉ còn 1 phần tử 'a'

# Ứng dụng: xóa trùng lặp khỏi list
numbers = [1, 2, 2, 3, 4, 3, 5]
unique  = list(set(numbers))
print(unique)   # [1, 2, 3, 4, 5]  (thứ tự không đảm bảo)
Mẹo hay: Chuyển list → set → list là cách nhanh nhất để xóa phần tử trùng lặp, nhưng thứ tự ban đầu không được bảo toàn.

4. Không có thứ tự cố định

Thứ tự phần tử khi duyệt set không xác định và có thể thay đổi giữa các lần chạy (phụ thuộc vào Python interpreter và phiên bản). Không thể truy cập phần tử set bằng index. 

my_set = {"a", "b", "c", "d"}

for item in my_set:
    print(item)
# Lần chạy 1: a b c d
# Lần chạy 2: d c a b  ← thứ tự có thể khác!

# LỖI: set không hỗ trợ index
# print(my_set[0])   # TypeError: 'set' object is not subscriptable
Lưu ý: Vì không có thứ tự, bạn không thể dùng index (set[0]) hay slicing với set. Nếu cần truy cập theo vị trí, hãy dùng list.

5. Thêm & xóa phần tử

Thêm phần tử – add()

my_set = {"a", "b", "c", "d"}

my_set.add("e")
print(my_set)   # {'a', 'b', 'c', 'd', 'e'}

# Thêm phần tử đã tồn tại → không có gì thay đổi
my_set.add("a")
print(my_set)   # {'a', 'b', 'c', 'd', 'e'}  — không lỗi, không thêm

Xóa phần tử – discard() và remove()

my_set = {"a", "b", "c", "d"}

# discard() – an toàn: không ném lỗi nếu phần tử không tồn tại
my_set.discard("a")
print(my_set)          # {'b', 'c', 'd'}

my_set.discard("z")    # phần tử không có → im lặng, không lỗi

# remove() – ném KeyError nếu phần tử không tồn tại
my_set.remove("b")
print(my_set)          # {'c', 'd'}

# my_set.remove("z")   # KeyError: 'z'
So sánh discard() và remove()
Phương thứcPhần tử không tồn tạiKhi nào dùng
discard(x) Bỏ qua, không lỗi Khi không chắc phần tử có trong set không
remove(x) KeyError Khi chắc chắn phần tử phải tồn tại

6. Duyệt qua set

Dùng vòng for — cú pháp giống list, nhưng thứ tự không đảm bảo:

my_set = {"a", "b", "c", "d"}

for item in my_set:
    print(item)
Nếu cần duyệt theo thứ tự nhất định, hãy dùng for item in sorted(my_set):.

7. Kiểm tra phần tử

Dùng in — tốc độ kiểm tra trên set nhanh hơn list (O(1) thay vì O(n)):

my_set = {"a", "b", "c"}

print("a" in my_set)    # True
print("z" in my_set)    # False
print("z" not in my_set)  # True

8. Giao (intersection)

Trả về tập hợp các phần tử thuộc cả hai set.

Set A
{1, 2, 3, 4}
∩
Set B
{3, 4, 5, 6}
→
Giao
{3, 4}
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Dùng phương thức
result = a.intersection(b)
print(result)   # {3, 4}

# Dùng toán tử &
result = a & b
print(result)   # {3, 4}

9. Hợp (union)

Trả về tập hợp chứa tất cả phần tử của cả hai set (không trùng lặp).

Set A
{1, 2, 3, 4}
∪
Set B
{3, 4, 5, 6}
→
Hợp
{1, 2, 3, 4, 5, 6}
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Dùng phương thức
result = a.union(b)
print(result)   # {1, 2, 3, 4, 5, 6}

# Dùng toán tử |
result = a | b
print(result)   # {1, 2, 3, 4, 5, 6}
Lưu ý: Cả union() lẫn toán tử | đều tạo ra set mới, không thay đổi set A hay set B. Nếu muốn cập nhật set A trực tiếp, dùng a.update(b) hoặc a |= b.

10. Hiệu (difference)

Trả về các phần tử chỉ thuộc set đứng trước, không có trong set kia. Kết quả phụ thuộc vào thứ tự.

Set A
{1, 2, 3, 4}
-
Set B
{3, 4, 5, 6}
→
A - B
{1, 2}
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# A - B: chỉ lấy phần tử thuộc A
diff1 = a.difference(b)
print(diff1)   # {1, 2}

# Toán tử -
diff1 = a - b
print(diff1)   # {1, 2}

# B - A: chỉ lấy phần tử thuộc B
diff2 = b - a
print(diff2)   # {5, 6}

11. Hiệu đối xứng (symmetric difference)

Trả về các phần tử chỉ thuộc một trong hai set — tức là phần của A không thuộc B, cộng với phần của B không thuộc A. Phần giao không được lấy.

Set A
{1, 2, 3, 4}
△
Set B
{3, 4, 5, 6}
→
Sym. Diff
{1, 2, 5, 6}
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# Phương thức
result = a.symmetric_difference(b)
print(result)   # {1, 2, 5, 6}

# Toán tử ^
result = a ^ b
print(result)   # {1, 2, 5, 6}

12. Toán tử tập hợp

Python cho phép dùng các ký hiệu toán học quen thuộc thay cho phương thức. Hai cách hoàn toàn tương đương:

Phương thức và toán tử tương ứng
Phép toánPhương thứcToán tửThay đổi set gốc?
Hợpa.union(b)a | bKhông
Cập nhật hợp vào Aa.update(b)a |= bCó (A)
Giaoa.intersection(b)a & bKhông
Hiệu A − Ba.difference(b)a - bKhông
Hiệu đối xứnga.symmetric_difference(b)a ^ bKhông
 
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)   # {1, 2, 3, 4, 5, 6}  — hợp
print(a & b)   # {3, 4}              — giao
print(a - b)   # {1, 2}              — hiệu A−B
print(b - a)   # {5, 6}              — hiệu B−A
print(a ^ b)   # {1, 2, 5, 6}        — hiệu đối xứng

# Kiểm tra: a và b không thay đổi
print(a)   # {1, 2, 3, 4}
print(b)   # {3, 4, 5, 6}

15. Bài tập thực hành

Bài tập 1: Xóa trùng lặp khỏi list

Cho list [1, 2, 2, 3, 4, 3, 5, 1]. Dùng set để loại bỏ phần tử trùng, sau đó chuyển về list và in ra.

numbers = [1, 2, 2, 3, 4, 3, 5, 1]
unique = list(set(numbers))
print(unique)   # thứ tự có thể khác [1, 2, 3, 4, 5]

Bài tập 2: Tìm bạn chung

Cho hai set bạn bè: friends_a = {"an", "binh", "cuong", "dung"} và friends_b = {"cuong", "dung", "em", "phuong"}. Tìm danh sách bạn chung của cả hai. 

friends_a = {"an", "binh", "cuong", "dung"}
friends_b = {"cuong", "dung", "em", "phuong"}

common = friends_a & friends_b
print("Bạn chung:", common)   # {'cuong', 'dung'}

Bài tập 3: Phân tích lớp học

Cho passed = {1, 2, 3, 4, 5} và failed = {4, 5, 6, 7, 8}. Tìm: (1) học sinh đỗ và trượt cùng lúc — không thể, nhưng hãy kiểm tra phần giao; (2) tất cả học sinh; (3) học sinh chỉ đỗ (không trượt).

Bài tập 4: Kiểm tra tập hợp con

Cho a = {1, 2} và b = {1, 2, 3, 4}. Kiểm tra a có phải tập hợp con của b không bằng issubset().

a = {1, 2}
b = {1, 2, 3, 4}
print(a.issubset(b))    # True
print(b.issuperset(a))  # True

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Set là kiểu mutable, không có thứ tự, không cho phép trùng lặp.
  • Tạo set bằng {a, b} (khi có phần tử) hoặc set() (khi rỗng). {} là dictionary rỗng.
  • Thêm phần tử: add(x). Xóa an toàn: discard(x). Xóa kiểm tra lỗi: remove(x).
  • Không dùng được index với set — không có set[0].
  • Phép toán tập hợp: hợp (|), giao (&), hiệu (-), hiệu đối xứng (^).
  • Tất cả phép toán trên đều tạo set mới và không thay đổi set gốc.
  • Kiểm tra phần tử với in trên set nhanh hơn list (O(1) vs O(n)).
Chúc mừng! Bạn đã hoàn thành cả bốn kiểu collection trong Python: list, tuple, dict và set. Đây là nền tảng quan trọng cho hầu hết mọi bài toán Python thực tế.
