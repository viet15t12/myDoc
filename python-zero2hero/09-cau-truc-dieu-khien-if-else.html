09. Cấu trúc điều khiển if / elif / else trong Python

Tài liệu học Python cơ bản về cấu trúc điều khiển if, elif, else: cú pháp, cách hoạt động từ trên xuống, toán tử logic, so sánh kép, if lồng nhau — kèm 5 ví dụ thực hành từ đơn giản đến phức tạp.

Giới thiệu

Trong lập trình, chúng ta thường cần kiểm tra một điều kiện rồi thực hiện hành động khác nhau tùy theo điều kiện đó đúng hay sai. Đây chính là chức năng của cấu trúc điều khiển.

Python cung cấp ba từ khoá cho cấu trúc này:

  • if — kiểm tra điều kiện đầu tiên.
  • elif (else if) — kiểm tra thêm điều kiện tiếp theo nếu điều kiện trước sai.
  • else — xử lý tất cả các trường hợp còn lại.
Ba từ khoá trong cấu trúc điều khiển Python
Từ khoá Khi nào dùng? Bắt buộc?
if Luôn luôn — đây là điều kiện đầu tiên Có
elif Khi cần kiểm tra thêm nhiều điều kiện khác Không
else Xử lý trường hợp không có điều kiện nào đúng Không

1. Cú pháp cơ bản

Cú pháp tổng quát của một cấu trúc điều khiển đầy đủ:

if <điều_kiện_1>:
    # Thực thi khi điều_kiện_1 đúng
elif <điều_kiện_2>:
    # Thực thi khi điều_kiện_1 sai và điều_kiện_2 đúng
elif <điều_kiện_3>:
    # Thực thi khi điều_kiện_1, 2 sai và điều_kiện_3 đúng
else:
    # Thực thi khi tất cả các điều kiện trên đều sai
Indentation (thụt lề): Python dùng thụt lề 4 dấu cách (hoặc 1 tab) để phân biệt các khối lệnh. Thiếu hoặc thừa thụt lề sẽ gây lỗi IndentationError.

2. Chỉ dùng if

Khi chỉ cần thực thi lệnh nếu điều kiện đúng, dùng mình if:

age = 20

if age >= 18:
    print("Bạn đã đủ tuổi trưởng thành.")

# Nếu age = 20 → in ra: "Bạn đã đủ tuổi trưởng thành."
# Nếu age = 15 → không in ra gì

3. if … else

Khi cần xử lý cả hai trường hợp đúng và sai:

number = int(input("Nhập số nguyên: "))

if number % 2 == 0:
    print(number, "là số chẵn.")
else:
    print(number, "là số lẻ.")

Flow hoạt động:

  • Nếu number = 10 → 10 % 2 == 0 đúng → in "10 là số chẵn."
  • Nếu number = 11 → 11 % 2 == 0 sai → vào else → in "11 là số lẻ."

4. if … elif … else

Khi có nhiều hơn hai trường hợp cần kiểm tra, dùng thêm elif:

score = float(input("Nhập điểm môn học (0–10): "))

if score > 8:
    print("Bạn đạt loại Giỏi.")
elif score > 6:
    print("Bạn đạt loại Khá.")
elif score > 4:
    print("Bạn đạt loại Trung bình.")
else:
    print("Bạn đạt loại Yếu.")
Python kiểm tra các điều kiện từ trên xuống dưới. Khi một điều kiện đúng, nó thực thi khối lệnh tương ứng rồi bỏ qua toàn bộ các nhánh còn lại — kể cả khi những nhánh dưới cũng đúng.

5. Kiểm tra từ trên xuống

Đây là nguyên tắc quan trọng nhất cần nhớ khi dùng if/elif/else. Hãy quan sát ví dụ sau với score = 7.5: 

score = 7.5

if score > 8:       # 7.5 > 8  → SAI → bỏ qua
    print("Giỏi")
elif score > 6:     # 7.5 > 6  → ĐÚNG → thực thi, dừng lại
    print("Khá")
else:               # Không bao giờ chạy đến đây
    print("Yếu")

# Output: Khá

Dù điều kiện score > 6 đúng, Python không tiếp tục kiểm tra else nữa. Toàn bộ cấu trúc kết thúc ngay khi tìm thấy điều kiện đúng đầu tiên.

Lỗi hay gặp — điều kiện chồng lấn: Tránh viết các điều kiện elif mà không xét điều kiện trước đó. Ví dụ sai: 
# SAI — điều kiện chồng lấn
if age < 6:
    print("Miễn phí")
if age < 60:        # ← if thứ hai TÁCH BIỆT, không phải elif
    print("70.000") # Tuổi = 4 vẫn in dòng này!
Dùng elif thay vì if thứ hai để tránh nhiều điều kiện cùng đúng.

6. Toán tử logic (and / or / not)

Kết hợp nhiều điều kiện trong một dòng bằng toán tử logic:

if 6 <= age and age < 18:
    print("Vé thiếu niên: 50.000đ")

# Tương đương cách viết dài hơn:
if age >= 6:

7. So sánh kép (chained comparison)

Python cho phép viết điều kiện kép ngắn gọn hơn so với nhiều ngôn ngữ khác:

# Cách thông thường
if 6 <= age and age < 18:
    ...

# Cách Python — so sánh kép (chained comparison)
if 6 <= age < 18:
    ...

# Cả hai cách cho kết quả hoàn toàn giống nhau
So sánh kép dễ đọc hơn và sát với ký hiệu toán học. Hãy dùng cách này khi kiểm tra một biến nằm trong một khoảng giá trị.

8. if lồng nhau (nested if)

Bạn có thể đặt một cấu trúc if bên trong một cấu trúc if khác:

year = int(input("Nhập năm: "))

if year % 4 == 0:               # chia hết cho 4?
    if year % 100 == 0 and year % 400 != 0:
        print(year, "không phải năm nhuận.")
    else:
        print(year, "là năm nhuận.")
else:
    print(year, "không phải năm nhuận.")
Nên hạn chế lồng quá nhiều cấp — code trở nên khó đọc. Khi có thể, hãy dùng and / or để gộp điều kiện lại trên một dòng.

9. Ví dụ: kiểm tra số chẵn / lẻ

Bài toán đơn giản nhất để làm quen với if/else: nhập một số nguyên và xác định chẵn hay lẻ.

number = int(input("Nhập số nguyên: "))

if number % 2 == 0:
    print("Số nhập vào", number, "là số chẵn.")
else:
    print("Số nhập vào", number, "là số lẻ.")

Kết quả chạy thử:

# Lần 1:
# Nhập số nguyên: 10
# → Số nhập vào 10 là số chẵn.

# Lần 2:
# Nhập số nguyên: 11
# → Số nhập vào 11 là số lẻ.

10. Ví dụ: xếp loại học lực

Nhập điểm từ 0–10 và in ra xếp loại tương ứng. Đây là ví dụ điển hình cho if/elif/else với nhiều nhánh.

score = float(input("Nhập điểm môn học (0–10): "))

if score > 8:
    print("Bạn đạt loại Giỏi.")
elif score > 6:
    print("Bạn đạt loại Khá.")
else:
    print("Bạn đạt loại Trung bình / Yếu.")

Bảng kết quả:

11. Ví dụ: giá vé theo tuổi

Nhập tuổi của người dùng và in ra giá vé vào công viên theo quy định:

  • Dưới 6 tuổi → Miễn phí
  • 6 đến dưới 18 tuổi → 50.000đ
  • 18 đến dưới 60 tuổi → 70.000đ
  • Từ 60 tuổi trở lên → 40.000đ
age = int(input("Nhập tuổi của bạn: "))

if age < 6:
    print("Giá vé: Miễn phí")
elif 6 <= age < 18:
    print("Giá vé: 50.000đ")
elif 18 <= age < 60:
    print("Giá vé: 70.000đ")
else:                           # age >= 60
    print("Giá vé: 40.000đ")

Kết quả chạy thử:

# Nhập tuổi của bạn: 4   → Giá vé: Miễn phí
# Nhập tuổi của bạn: 15  → Giá vé: 50.000đ
# Nhập tuổi của bạn: 21  → Giá vé: 70.000đ
# Nhập tuổi của bạn: 65  → Giá vé: 40.000đ
Lưu ý thực tế: Chương trình này không xử lý trường hợp người dùng nhập chữ hoặc số âm. Khi học đến vòng lặp và xử lý ngoại lệ (exception), bạn có thể quay lại cải tiến phần này.

12. Ví dụ: kiểm tra năm nhuận

Quy tắc xác định năm nhuận:

  • Chia hết cho 4 → có thể là năm nhuận.
  • Nhưng nếu chia hết cho 100 mà không chia hết cho 400 → không phải năm nhuận.
  • Ví dụ: 2000 là năm nhuận (chia hết 400); 1900 không phải (chia hết 100 nhưng không chia hết 400); 2024 là năm nhuận (chia hết 4, không chia hết 100).
year = int(input("Nhập năm: "))

if year % 4 == 0:
    # Chia hết 4 — kiểm tra thêm ngoại lệ 100 và 400
    if year % 100 == 0 and year % 400 != 0:
        print(year, "không phải năm nhuận.")
    else:
        print(year, "là năm nhuận.")
else:
    # Không chia hết 4 → chắc chắn không phải năm nhuận
    print(year, "không phải năm nhuận.")

19000 ✓0 ✓300 ✗Không nhuận
20000 ✓0 ✓0 ✓Năm nhuận

13. Ví dụ: chuyển đổi nhiệt độ

Cho phép người dùng chọn chiều chuyển đổi rồi thực hiện tính toán:

  • Nhập 1 → chuyển từ °F sang °C
  • Nhập 2 → chuyển từ °C sang °F
print("Nhập yêu cầu của bạn:")
print("1. Chuyển đổi từ độ F sang độ C")
print("2. Chuyển đổi từ độ C sang độ F")
choice = input("Lựa chọn của bạn: ")

if choice == "1":
    # F → C
    f = float(input("Nhập độ F: "))
    c = (f - 32) / 1.8
    print("Độ C nhận được:", c)

if choice == "2":
    # C → F
    c = float(input("Nhập độ C: "))
    f = c * 1.8 + 32
    print("Độ F nhận được:", f)

Kết quả chạy thử:

# Lựa chọn: 1 → Nhập độ F: 30 → Độ C nhận được: -1.111...
# Lựa chọn: 2 → Nhập độ C: 20 → Độ F nhận được: 68.0
Công thức:
°C = (°F − 32) ÷ 1.8
°F = °C × 1.8 + 32
Trong ví dụ này, hai khối if được viết tách biệt nhau (không dùng elif). Điều này vẫn đúng vì choice không thể đồng thời bằng "1" và "2". Tuy nhiên, dùng if/elif là best practice và hiệu quả hơn vì tránh kiểm tra điều kiện thừa.

16. Bài tập thực hành

Bài tập 1: Số dương, âm hay bằng 0

Nhập một số thực. In ra "Số dương", "Số âm" hoặc "Bằng 0" tương ứng.

num = float(input("Nhập số: "))

if num > 0:
    print("Số dương")
elif num < 0:
    print("Số âm")
else:
    print("Bằng 0")

Bài tập 2: Xếp loại học sinh chi tiết hơn

Mở rộng ví dụ xếp loại để có 4 mức: Xuất sắc (>9), Giỏi (>8), Khá (>6), Trung bình (>4), Yếu (còn lại).

Bài tập 3: Máy tính đơn giản

Nhập hai số và một phép tính (+, -, *, /). Dùng if/elif/else để tính và in kết quả. Lưu ý xử lý trường hợp chia cho 0. 

a = float(input("Nhập số a: "))
b = float(input("Nhập số b: "))
op = input("Phép tính (+, -, *, /): ")

if op == "+":
    print("Kết quả:", a + b)
elif op == "-":
    print("Kết quả:", a - b)
elif op == "*":
    print("Kết quả:", a * b)
elif op == "/":
    if b == 0:
        print("Lỗi: không thể chia cho 0.")
    else:
        print("Kết quả:", a / b)
else:
    print("Phép tính không hợp lệ.")

Bài tập 4: Ngày trong tuần

Nhập vào một số từ 1 đến 7. In ra tên ngày tương ứng (1 = Thứ Hai, …, 7 = Chủ Nhật). Nếu số ngoài khoảng, in "Không hợp lệ".

Bài tập 5: BMI và phân loại

Nhập cân nặng (kg) và chiều cao (m). Tính BMI = cân nặng / chiều cao². Dùng if/elif/else phân loại: Thiếu cân (BMI < 18.5), Bình thường (18.5–24.9), Thừa cân (25–29.9), Béo phì (≥ 30).

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • if luôn là bước đầu — kiểm tra điều kiện đầu tiên.
  • elif kiểm tra điều kiện tiếp theo khi điều kiện trước sai. Có thể có nhiều elif.
  • else xử lý tất cả trường hợp còn lại — không có điều kiện đi kèm.
  • Python kiểm tra từ trên xuống dưới. Khi điều kiện nào đúng, thực thi rồi bỏ qua hết những nhánh bên dưới.
  • Kết hợp điều kiện bằng and, or, not.
  • So sánh kép 6 <= age < 18 là đặc trưng của Python — ngắn gọn và dễ đọc.
  • Hạn chế lồng if quá nhiều cấp. Ưu tiên dùng and/or để đơn giản hóa.
Chúc mừng! Bạn đã nắm vững cấu trúc điều khiển — nền tảng để xây dựng mọi chương trình có logic phân nhánh. Bài tiếp theo chúng ta sẽ học về vòng lặp (for và while) để xử lý các tác vụ lặp đi lặp lại.
