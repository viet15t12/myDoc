07. Kiểu dữ liệu Dictionary trong Python

Tài liệu học Python cơ bản về kiểu dữ liệu dict: khởi tạo, truy cập phần tử bằng key, thêm / cập nhật / xóa phần tử, duyệt qua keys – values – items, và dictionary comprehension.

Giới thiệu

Dictionary (dict) là một trong bốn kiểu collection của Python. Dictionary lưu dữ liệu dưới dạng ánh xạ (map) giữa key và value — mỗi key tương ứng với một value duy nhất.

So sánh nhanh 4 kiểu collection phổ biến trong Python
Kiểu dữ liệu Ký hiệu Thay đổi được? Có thứ tự? Cho phép trùng lặp?
listDanh sách linh hoạt [...] Có Có Có
tupleDanh sách cố định (...) Không Có Có
setTập hợp giá trị duy nhất {...} Có Không Không
dictCặp khóa – giá trị {k: v} Có Có Không trùng key

Dictionary là kiểu mutable và có thứ tự (từ Python 3.7+, thứ tự chèn được bảo toàn). Key phải là kiểu immutable (string, int, tuple…), còn value có thể là bất kỳ kiểu nào.

1. Tạo dictionary

Dùng cặp dấu ngoặc nhọn {}. Mỗi cặp key–value được ngăn cách nhau bằng dấu hai chấm :, các cặp cách nhau bằng dấu phẩy. 

# Cú pháp: {key: value, key: value, ...}
name_age = {"min": 5, "na": 6, "bong": 10}

print(name_age)
print(type(name_age))

Kết quả:

{'min': 5, 'na': 6, 'bong': 10}
<class 'dict'>
Key phải là immutable: string, int, float, tuple đều dùng được làm key. List không thể làm key vì list là mutable.

Bạn cũng có thể tạo dictionary bằng hàm dict() với keyword arguments:

# Dùng hàm dict()
name_age = dict(min=5, na=6, bong=10)
print(name_age)   # {'min': 5, 'na': 6, 'bong': 10}
Lưu ý: Khi dùng dict() theo cách trên, key phải là identifier Python hợp lệ (không có khoảng trắng, không bắt đầu bằng số).

2. Dictionary rỗng

Có hai cách tạo dictionary rỗng:

# Cách 1: cặp ngoặc nhọn rỗng
empty1 = {}

# Cách 2: hàm dict()
empty2 = dict()

print(type(empty1))   # <class 'dict'>
print(type(empty2))   # <class 'dict'>
Cẩn thận – dễ nhầm với set! {} tạo ra dictionary rỗng, không phải set rỗng. Muốn tạo set rỗng phải dùng set().

3. Truy cập phần tử

Dùng cú pháp dict[key] để lấy value tương ứng với key đó. 

name_age = {"min": 5, "na": 6, "bong": 10}

print(name_age["na"])     # 6
print(name_age["bong"])   # 10

Kết quả:

6
10
KeyError: Truy cập một key không tồn tại sẽ ném ngay KeyError. Dùng get() (xem phần 4) nếu bạn không chắc key có tồn tại không. 
name_age = {"min": 5, "na": 6}
print(name_age["xuan"])   # KeyError: 'xuan'

4. Phương thức get()

dict.get(key) trả về value nếu key tồn tại, ngược lại trả về None — thay vì ném KeyError. Bạn có thể cung cấp giá trị mặc định ở tham số thứ hai. 

name_age = {"min": 5, "na": 6, "bong": 10}

age = name_age.get("na")
print(age)                         # 6

age_unknown = name_age.get("xuan")
print(age_unknown)                 # None

age_default = name_age.get("xuan", 0)
print(age_default)                 # 0
Thực tế: Khi xử lý dữ liệu không chắc chắn (đọc từ file, API…), dùng get() an toàn hơn nhiều so với dict[key].

5. Thêm & cập nhật phần tử

Cú pháp gán dict[key] = value hoạt động theo hai trường hợp:

  • Nếu key chưa có → thêm cặp key–value mới vào cuối dictionary.
  • Nếu key đã có → cập nhật (ghi đè) value cũ.
name_age = {"min": 5, "na": 6, "bong": 10}

# Cập nhật: bong tăng lên 12 tuổi sau 2 năm
name_age["bong"] = 12

# Thêm mới: nam 30 tuổi
name_age["nam"] = 30

print(name_age)

Kết quả:

{'min': 5, 'na': 6, 'bong': 12, 'nam': 30}

Dùng update() để thêm / cập nhật nhiều cặp cùng lúc:

name_age.update({"lan": 22, "bong": 13})
print(name_age)
# {'min': 5, 'na': 6, 'bong': 13, 'nam': 30, 'lan': 22}

6. Xóa phần tử

Python cung cấp nhiều cách xóa phần tử khỏi dictionary:

name_age = {"min": 5, "na": 6, "bong": 10, "nam": 30}

# del – xóa theo key, ném KeyError nếu không tồn tại
del name_age["na"]
print(name_age)   # {'min': 5, 'bong': 10, 'nam': 30}

# pop(key) – xóa và trả về value
age = name_age.pop("bong")
print(age)         # 10
print(name_age)    # {'min': 5, 'nam': 30}

# pop(key, default) – an toàn, không ném KeyError
val = name_age.pop("ghost", None)
print(val)         # None

# clear() – xóa toàn bộ
name_age.clear()
print(name_age)    # {}
So sánh các cách xóa phần tử
Cú phápTrả vềKey không tồn tại
del dict[key]Không trả vềKeyError
dict.pop(key)Value đã xóaKeyError
dict.pop(key, default)Value hoặc defaultTrả về default
dict.clear()Không trả vềN/A – xóa tất cả

7. Duyệt qua keys

Dùng vòng for trực tiếp trên dictionary (mặc định duyệt qua các key), hoặc gọi rõ phương thức keys(). 

name_age = {"min": 5, "na": 6, "bong": 10}

# Cách 1: for k in dict
for k in name_age:
    print(k)

# Cách 2: for k in dict.keys()  — tương đương
for k in name_age.keys():
    print(k)

Kết quả (cả hai cách giống nhau):

min
na
bong

8. Duyệt qua values

Dùng dict.values():

name_age = {"min": 5, "na": 6, "bong": 10}

for v in name_age.values():
    print(v)

Kết quả:

5
6
10

9. Duyệt qua items

Dùng dict.items() để lấy cả key và value trong cùng một lần lặp — cách được dùng phổ biến nhất trong thực tế. 

name_age = {"min": 5, "na": 6, "bong": 10}

for k, v in name_age.items():
    print(k, v)

Kết quả:

min 5
na 6
bong 10
Ghi nhớ: items() trả về chuỗi các tuple (key, value). Khi dùng for k, v in ..., Python tự động giải nén (unpack) mỗi tuple thành hai biến.

10. Kiểm tra key tồn tại

Dùng toán tử in để kiểm tra xem một key có trong dictionary không:

name_age = {"min": 5, "na": 6, "bong": 10}

print("na" in name_age)      # True
print("xuan" in name_age)    # False
print("xuan" not in name_age) # True

# Ứng dụng thực tế: truy cập an toàn
key = "na"
if key in name_age:
    print(f"{key} có trong dictionary, tuổi: {name_age[key]}")
else:
    print(f"{key} không tồn tại")

11. Dictionary comprehension

Tương tự list comprehension, bạn có thể tạo dictionary từ một iterable bằng cú pháp ngắn gọn: 

# Tạo dictionary: số -> bình phương của số
squares = {x: x**2 for x in range(1, 6)}
print(squares)
# {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# Tạo dictionary từ hai list
names  = ["min", "na", "bong"]
ages   = [5, 6, 10]
name_age = {k: v for k, v in zip(names, ages)}
print(name_age)
# {'min': 5, 'na': 6, 'bong': 10}

# Lọc: chỉ giữ những người dưới 8 tuổi
young = {k: v for k, v in name_age.items() if v < 8}
print(young)
# {'min': 5, 'na': 6}

14. Bài tập thực hành

Bài tập 1: Đếm tần suất ký tự

Cho chuỗi "hello world". Đếm số lần xuất hiện của mỗi ký tự (bỏ qua khoảng trắng) và lưu vào dictionary.

text = "hello world"
freq = {}
for ch in text:
    if ch == " ":
        continue
    freq[ch] = freq.get(ch, 0) + 1
print(freq)
# {'h': 1, 'e': 1, 'l': 3, 'o': 2, 'w': 1, 'r': 1, 'd': 1}

Bài tập 2: Đảo ngược dictionary

Cho {"a": 1, "b": 2, "c": 3}. Tạo dictionary mới với key và value đổi chỗ nhau.

original = {"a": 1, "b": 2, "c": 3}
reversed_dict = {v: k for k, v in original.items()}
print(reversed_dict)   # {1: 'a', 2: 'b', 3: 'c'}

Bài tập 3: Gộp hai dictionary

Cho hai dictionary d1 = {"x": 1, "y": 2} và d2 = {"y": 99, "z": 3}. Gộp chúng lại sao cho key trùng lặp lấy giá trị của d2.

Bài tập 4: Thống kê điểm thi

Nhập vào danh sách điểm thi của một lớp. In ra điểm cao nhất, thấp nhất và điểm trung bình. Lưu kết quả vào một dictionary {"max": ..., "min": ..., "avg": ...}.

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Dictionary lưu dữ liệu dạng cặp key – value, ký hiệu {key: value}.
  • Key phải là immutable; mỗi key là duy nhất trong dictionary.
  • Tạo dictionary bằng {} hoặc dict(). Dictionary rỗng: {} hoặc dict().
  • Truy cập: dict[key] (có thể raise KeyError) hoặc dict.get(key, default) (an toàn hơn).
  • Thêm / cập nhật: dict[key] = value. Xóa: del dict[key] hoặc pop(key).
  • Duyệt: keys(), values(), items() — dùng items() khi cần cả key lẫn value.
  • Dictionary comprehension giúp tạo dict ngắn gọn từ iterable.
Bài tiếp theo: Set — kiểu collection chứa các phần tử không trùng lặp và không có thứ tự cố định. Set rất hữu ích cho bài toán tập hợp (giao, hợp, hiệu).
