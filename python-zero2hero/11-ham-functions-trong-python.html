11. Hàm (Functions) trong Python

Tài liệu học Python cơ bản về hàm: cách định nghĩa với def, tham số, giá trị trả về, tham số mặc định, *args / **kwargs, phạm vi biến (scope), hàm lambda, và các hàm built-in thông dụng. Có ví dụ thực tế copy file và bài tập thực hành.

Giới thiệu

Khi lập trình, rất nhiều đoạn code cần được dùng lại ở nhiều chỗ khác nhau. Thay vì copy-paste cùng một đoạn code, bạn đóng gói nó thành một hàm — đặt tên, gọi khi cần, và có thể truyền dữ liệu vào qua tham số.

Không dùng hàmDùng hàm
Copy-paste code → khó bảo trì, dễ sót lỗi Viết một lần → gọi nhiều lần ở bất kỳ đâu
Khi sửa phải sửa ở từng nơi Sửa trong hàm → áp dụng toàn bộ chương trình
Code dài, khó đọc Code ngắn gọn, mỗi hàm giải quyết một việc rõ ràng
Nguyên tắc DRY: Don't Repeat Yourself — Đừng lặp lại chính mình. Hàm là công cụ chính để tuân thủ nguyên tắc này trong Python.

1. Định nghĩa hàm với def

Cú pháp định nghĩa hàm trong Python dùng từ khóa def: 

def ten_ham(tham_so_1, tham_so_2):
    # thân hàm (thụt vào 4 dấu cách)
    ket_qua = tham_so_1 + tham_so_2
    return ket_qua

Ví dụ hàm đơn giản nhất — không tham số, không return:

def say_hello():
    print("Xin chào Python!")

say_hello()   # Xin chào Python!

2. Gọi hàm

Định nghĩa hàm chưa chạy bất kỳ lệnh nào. Bạn phải gọi hàm (hay còn gọi là invoke) bằng cách viết tên hàm kèm dấu ngoặc tròn: 

def greet(name):
    print(f"Xin chào, {name}!")

# Gọi hàm
greet("An")       # Xin chào, An!
greet("Bình")     # Xin chào, Bình!
greet("Python")   # Xin chào, Python!
Lưu ý thứ tự: Bạn phải định nghĩa hàm trước khi gọi nó. Gọi hàm trước khi định nghĩa sẽ gây lỗi NameError. 
# [SAI] Gọi trước khi định nghĩa
greet("An")

def greet(name):
    print(f"Xin chào, {name}!")

# [DUNG] Định nghĩa trước, gọi sau
def greet(name):
    print(f"Xin chào, {name}!")

greet("An")

3. Parameter vs Argument

Hai thuật ngữ này thường bị dùng lẫn lộn, nhưng chúng có nghĩa khác nhau:

Thuật ngữTiếng ViệtLà gì?Ở đâu?
Parameter Tham số Biến được khai báo trong dấu () khi định nghĩa hàm Trong def
Argument Đối số Giá trị thực tế truyền vào khi gọi hàm Khi gọi hàm 
#          x, y  ← parameter (tham số)
def tinh_tong(x, y):
    return x + y

a = 6
b = 5
#                  a, b  ← argument (đối số)
ket_qua = tinh_tong(a, b)
print(ket_qua)   # 11

4. Giá trị trả về – return

Dùng return để hàm trả về một giá trị cho người gọi. Khi Python gặp return, nó thoát khỏi hàm ngay lập tức.

Hàm có return

def binh_phuong(x):
    return x * x

ket_qua = binh_phuong(5)
print(ket_qua)   # 25
print(binh_phuong(3) + binh_phuong(4))   # 25

Hàm không có return

def in_thong_tin(ten, tuoi):
    print(f"Tên: {ten}, Tuổi: {tuoi}")

in_thong_tin("An", 22)   # Tên: An, Tuổi: 22

# Nếu cố gán kết quả, sẽ nhận về None
gia_tri = in_thong_tin("Bình", 25)
print(gia_tri)   # None

Return nhiều giá trị

Python cho phép return nhiều giá trị cùng lúc — thực chất là trả về một tuple:

def chia_lay_du(a, b):
    thuong = a // b
    du = a % b
    return thuong, du

thuong, du = chia_lay_du(17, 5)
print(f"17 ÷ 5 = {thuong} dư {du}")   # 17 ÷ 5 = 3 dư 2

Return sớm để kiểm tra điều kiện

def chia_an_toan(a, b):
    if b == 0:
        return None   # thoát sớm, tránh lỗi ZeroDivisionError
    return a / b

print(chia_an_toan(10, 2))   # 5.0
print(chia_an_toan(10, 0))   # None
Quy tắc: Hàm dùng để tính toán / xử lý và trả kết quả → dùng return. Hàm dùng để thực hiện hành động (in, ghi file…) → không cần return.

5. Tham số mặc định (Default parameter)

Bạn có thể gán giá trị mặc định cho tham số. Nếu người gọi không truyền đối số cho tham số đó, Python dùng giá trị mặc định. 

def chao_hoi(ten, loi_chao="Xin chào"):
    print(f"{loi_chao}, {ten}!")

chao_hoi("An")               # Xin chào, An!
chao_hoi("Bình", "Chào buổi sáng")   # Chào buổi sáng, Bình!
Quy tắc quan trọng: Tham số có giá trị mặc định phải đặt sau tham số không có giá trị mặc định. Đặt ngược lại sẽ bị SyntaxError. 
# [SAI]
def ham_sai(a=1, b):
    pass

# [DUNG]
def ham_dung(a, b=1):
    pass

6. Keyword argument

Khi gọi hàm, bạn có thể truyền đối số theo tên tham số — gọi là keyword argument. Cách này giúp code dễ đọc hơn và không cần quan tâm đến thứ tự. 

def mo_ta_nguoi(ten, tuoi, thanh_pho):
    print(f"{ten}, {tuoi} tuổi, sống ở {thanh_pho}")

# Positional argument — phải đúng thứ tự
mo_ta_nguoi("An", 22, "Hà Nội")

# Keyword argument — không cần đúng thứ tự
mo_ta_nguoi(tuoi=25, thanh_pho="TP.HCM", ten="Bình")

# Trộn positional và keyword — positional phải đặt trước
mo_ta_nguoi("Châu", tuoi=30, thanh_pho="Đà Nẵng")
An, 22 tuổi, sống ở Hà Nội
Bình, 25 tuổi, sống ở TP.HCM
Châu, 30 tuổi, sống ở Đà Nẵng

7. *args và **kwargs

Khi không biết trước sẽ nhận bao nhiêu đối số, Python cung cấp hai cú pháp đặc biệt:

*args — nhận nhiều positional argument

*args gom tất cả positional argument thừa vào một tuple. 

def tinh_tong(*so):
    print(type(so))   # <class 'tuple'>
    return sum(so)

print(tinh_tong(1, 2))           # 3
print(tinh_tong(1, 2, 3, 4, 5)) # 15

**kwargs — nhận nhiều keyword argument

**kwargs gom tất cả keyword argument thừa vào một dict. 

def in_thong_tin(**thong_tin):
    for key, value in thong_tin.items():
        print(f"  {key}: {value}")

in_thong_tin(ten="An", tuoi=22, nghe_nghiep="Lập trình viên")
  ten: An
  tuoi: 22
  nghe_nghiep: Lập trình viên

8. Phạm vi biến – Scope

Scope xác định nơi một biến có thể được truy cập. Python có hai scope chính:

ScopeĐịnh nghĩa ở đâuTruy cập được từ đâu
Local (cục bộ) Bên trong hàm Chỉ trong hàm đó
Global (toàn cục) Bên ngoài mọi hàm Mọi nơi trong file 
x = 10   # biến global

def ham_vi_du():
    y = 20   # biến local
    print(x)  # đọc được biến global
    print(y)  # đọc được biến local

ham_vi_du()
print(x)  # 10
print(y)  # NameError — y không tồn tại ngoài hàm

Từ khóa global

Nếu bạn muốn sửa biến global từ bên trong hàm, dùng từ khóa global:

dem = 0

def tang_dem():
    global dem
    dem += 1

tang_dem()
tang_dem()
print(dem)   # 2
Thực hành tốt: Hạn chế dùng global. Thay vào đó, hãy truyền giá trị qua tham số và nhận kết quả qua return — code dễ debug hơn nhiều.

9. Hàm lồng nhau (Nested function)

Python cho phép định nghĩa hàm bên trong một hàm khác. Hàm bên trong chỉ có thể dùng trong phạm vi hàm ngoài. 

def ham_ngoai(ten):
    def chao():            # hàm bên trong
        return "Xin chào"
    ket_qua = f"{chao()}, {ten}!"
    return ket_qua

print(ham_ngoai("An"))   # Xin chào, An!

# chao() không thể gọi từ ngoài ham_ngoai
Hàm lồng nhau hay dùng khi bạn muốn tổ chức logic nội bộ của một hàm phức tạp mà không muốn expose ra ngoài.

10. Hàm lambda

lambda là cách viết hàm ngắn gọn, không tên trên một dòng. Phù hợp cho các hàm đơn giản, thường dùng như đối số cho các hàm khác. 

# Cú pháp: lambda tham_so: bieu_thuc_tra_ve
nhan_doi = lambda x: x * 2
print(nhan_doi(5))   # 10

# Tương đương với:
def nhan_doi(x):
    return x * 2

Lambda với nhiều tham số

tong = lambda a, b: a + b
print(tong(3, 4))   # 7

Lambda kết hợp với sorted() và map()

# Sắp xếp list dict theo một field
hoc_sinh = [
    {"ten": "An",   "diem": 8.5},
    {"ten": "Bình", "diem": 9.0},
    {"ten": "Châu", "diem": 7.5},
]
sorted_hs = sorted(hoc_sinh, key=lambda hs: hs["diem"], reverse=True)
for hs in sorted_hs:
    print(hs["ten"], "–", hs["diem"])
Bình – 9.0
An – 8.5
Châu – 7.5
# map() — áp dụng hàm cho từng phần tử
so = [1, 2, 3, 4, 5]
binh_phuong = list(map(lambda x: x**2, so))
print(binh_phuong)   # [1, 4, 9, 16, 25]
Khi nào dùng lambda? Khi hàm chỉ cần một dòng, dùng như đối số tạm thời (sorted, map, filter). Nếu logic phức tạp hơn, nên dùng def cho rõ ràng.

12. Ví dụ thực tế: copy file

Dưới đây là ví dụ kết hợp hàm với thư viện shutil và os để copy toàn bộ file từ một folder sang folder khác — một tác vụ rất hay gặp trong thực tế.

Bước 1: Không dùng hàm (code thẳng)

import shutil
import os

source_dir = "source_d"
destination_dir = "destination_d"

list_name = os.listdir(source_dir)

for file_name in list_name:
    shutil.copy(
        os.path.join(source_dir, file_name),
        destination_dir
    )

Bước 2: Đóng gói vào hàm (tái sử dụng được)

import shutil
import os

def copy_all_files(source_dir, destination_dir):
    """Copy tất cả file từ source_dir sang destination_dir."""
    list_name = os.listdir(source_dir)
    for file_name in list_name:
        shutil.copy(
            os.path.join(source_dir, file_name),
            destination_dir
        )
    print(f"Đã copy {len(list_name)} file thành công.")

# Gọi hàm — bây giờ có thể dùng lại ở bất kỳ đâu
copy_all_files("source_d", "destination_d")
copy_all_files("photos", "photos_backup")   # dùng lại với folder khác
Nhận xét: Bằng cách đóng gói vào hàm, cùng logic copy file nay có thể dùng cho bất kỳ cặp folder nào — chỉ cần truyền vào tham số khác nhau. Đây chính là sức mạnh của hàm.

Bước 3: Thêm xử lý lỗi

import shutil
import os

def copy_all_files(source_dir, destination_dir):
    """Copy tất cả file từ source_dir sang destination_dir."""
    if not os.path.exists(source_dir):
        print(f"Thư mục nguồn '{source_dir}' không tồn tại!")
        return

    os.makedirs(destination_dir, exist_ok=True)  # tạo thư mục đích nếu chưa có

    list_name = os.listdir(source_dir)
    for file_name in list_name:
        shutil.copy(
            os.path.join(source_dir, file_name),
            destination_dir
        )
    print(f"Đã copy {len(list_name)} file sang '{destination_dir}'")

copy_all_files("source_d", "destination_d")

13. Ví dụ thực tế: máy tính đơn giản

Bài toán: xây dựng một chương trình máy tính nhỏ cho phép cộng, trừ, nhân, chia — mỗi phép tính là một hàm riêng. 

def cong(a, b):
    return a + b

def tru(a, b):
    return a - b

def nhan(a, b):
    return a * b

def chia(a, b):
    if b == 0:
        return "Lỗi: không thể chia cho 0"
    return a / b

def tinh_toan(a, phep_tinh, b):
    """Hàm điều phối: chọn phép tính dựa theo chuỗi."""
    if phep_tinh == "+":
        return cong(a, b)
    elif phep_tinh == "-":
        return tru(a, b)
    elif phep_tinh == "*":
        return nhan(a, b)
    elif phep_tinh == "/":
        return chia(a, b)
    else:
        return "Phép tính không hợp lệ"

# Sử dụng
print(tinh_toan(10, "+", 5))   # 15
print(tinh_toan(10, "/", 0))   # Lỗi: không thể chia cho 0
print(tinh_toan(7,  "*", 6))   # 42

16. Bài tập thực hành

Bài tập 1: Hàm tính chu vi và diện tích hình chữ nhật

Viết hàm hinh_chu_nhat(dai, rong) trả về tuple (chu_vi, dien_tich). Gọi hàm với dai=8, rong=5 và in kết quả.

def hinh_chu_nhat(dai, rong):
    chu_vi = 2 * (dai + rong)
    dien_tich = dai * rong
    return chu_vi, dien_tich

cv, dt = hinh_chu_nhat(8, 5)
print(f"Chu vi: {cv}, Diện tích: {dt}")   # Chu vi: 26, Diện tích: 40

Bài tập 2: Hàm kiểm tra số nguyên tố

Viết hàm la_so_nguyen_to(n) trả về True nếu n là số nguyên tố, False nếu không. Dùng hàm này để in tất cả số nguyên tố từ 2 đến 50.

def la_so_nguyen_to(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

primes = [n for n in range(2, 51) if la_so_nguyen_to(n)]
print(primes)

Bài tập 3: Hàm đếm từ

Viết hàm dem_tu(chuoi) nhận một chuỗi và trả về dict đếm số lần xuất hiện của mỗi từ. Không phân biệt chữ hoa/thường.

def dem_tu(chuoi):
    tu_dien = {}
    for tu in chuoi.lower().split():
        tu_dien[tu] = tu_dien.get(tu, 0) + 1
    return tu_dien

print(dem_tu("Python is great and Python is fun"))
# {'python': 2, 'is': 2, 'great': 1, 'and': 1, 'fun': 1}

Bài tập 4: Hàm với *args — tìm giá trị lớn nhất

Viết hàm tim_max(*so) dùng *args để nhận số lượng số bất kỳ và trả về số lớn nhất. Không dùng hàm built-in max().

Bài tập 5: Sắp xếp danh sách sinh viên

Có danh sách sinh viên sau. Dùng sorted() kết hợp lambda để sắp xếp theo điểm từ cao xuống thấp, nếu bằng điểm thì sắp xếp tên theo bảng chữ cái.

sinh_vien = [
    {"ten": "An",   "diem": 8.5},
    {"ten": "Bình", "diem": 9.0},
    {"ten": "Châu", "diem": 8.5},
    {"ten": "Dũng", "diem": 7.0},
]
# Gợi ý: key=lambda sv: (-sv["diem"], sv["ten"])

Tổng kết

Sau bài này, bạn cần nắm chắc:

  • Dùng def ten_ham(tham_so): để định nghĩa hàm; phải định nghĩa trước khi gọi.
  • Parameter là biến trong def; argument là giá trị truyền vào lúc gọi.
  • return trả về giá trị và kết thúc hàm; hàm không có return trả về None.
  • Tham số mặc định giúp hàm linh hoạt hơn; tham số không có default phải đứng trước.
  • Keyword argument giúp code dễ đọc và không cần quan tâm thứ tự tham số.
  • *args gom positional argument thừa vào tuple; **kwargs gom keyword argument thừa vào dict.
  • Biến local chỉ sống trong hàm; biến global truy cập được mọi nơi — hạn chế dùng global.
  • lambda x: x*2 là hàm ẩn danh một dòng, thường dùng làm đối số cho sorted, map, filter.
