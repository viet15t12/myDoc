# myDoc

## Template chung cho tài liệu

Thư mục `docs-template/` là mẫu chung dựa trên cấu trúc của `lpic-101-102`.

Cách dùng:

1. Sao chép toàn bộ `docs-template/` sang thư mục mới.
2. Chỉnh `index.html`, `lessons.json`, `quick-links.json` theo nội dung của tài liệu mới.
3. Nếu cần, đổi `VISITED_KEY` trong `doc-script.js` để tránh trùng bộ tài liệu khác.
