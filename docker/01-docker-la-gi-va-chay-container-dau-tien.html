01. Docker là gì và chạy container đầu tiên

Bài mở đầu giúp bạn hiểu Docker giải quyết vấn đề gì, image khác container thế nào và cách chạy container đầu tiên.

Mục tiêu

Mục tiêu bài học: hiểu khái niệm image/container, chạy hello-world và biết một vài lệnh Docker cơ bản.
  • Hiểu container đóng gói ứng dụng cùng môi trường chạy.
  • Phân biệt image và container.
  • Biết chạy, xem danh sách và xóa container.

1. Docker là gì?

Docker là nền tảng giúp đóng gói ứng dụng vào container. Container nhẹ hơn máy ảo vì dùng chung kernel của hệ điều hành host, nhưng vẫn cô lập tiến trình, filesystem và network ở mức cần thiết.

2. Image và container

Thành phầnCách hiểu nhanh
ImageBản mẫu chỉ đọc, chứa app và dependency.
ContainerMột tiến trình đang chạy được tạo từ image.
RegistryNơi lưu image, ví dụ Docker Hub.

3. Lệnh đầu tiên

Sau khi cài Docker, chạy lệnh kiểm tra:

docker run hello-world

Docker sẽ tải image nếu máy chưa có, tạo container, chạy chương trình rồi dừng lại.

4. Lệnh cơ bản

docker version
docker images
docker ps
docker ps -a
docker rm container_id
docker rmi image_id

5. Thực hành

Chạy Nginx trong container và mở trình duyệt tại http://localhost:8080.

docker run --name web-demo -p 8080:80 nginx
Lưu ý: nếu cổng 8080 đang được ứng dụng khác dùng, đổi sang cổng khác như 8081:80.

Tổng kết

Docker giúp môi trường chạy nhất quán giữa máy dev, CI và server. Bước tiếp theo là học Dockerfile để tự build image cho ứng dụng của bạn.
