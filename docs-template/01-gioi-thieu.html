Bài mẫu đầu tiên

Bài này là khung tham khảo để tạo các bài học mới có mục lục, ví dụ, ghi chú và điều hướng.

Mục tiêu

Mục tiêu bài học: dùng file này làm mẫu cấu trúc khi tạo bài học mới.

Sau bài này, bạn biết cách nhân bản template thành một lộ trình mới, cập nhật metadata và viết bài đầu tiên.

  • Biết vai trò của index.html, lessons.json, style.css và doc-styles.css.
  • Biết cách đặt tên file bài học theo thứ tự.
  • Biết cách tạo mục lục khớp với các section id.

1. Cấu trúc bài học

Mỗi khóa học nên có một trang danh sách bài và nhiều trang bài học riêng.

ten-khoa-hoc/
├── index.html
├── lessons.json
├── style.css
├── doc-styles.css
├── doc-script.js
└── 01-gioi-thieu.html

2. Dữ liệu trong lessons.json

Trang chủ đọc lessons.json để hiển thị tiêu đề khóa học, danh mục và danh sách bài.

{
  "id": "01",
  "type": "basic",
  "title": "Bài mẫu đầu tiên",
  "file": "01-gioi-thieu.html",
  "status": "published"
}

3. Ví dụ code

Khối code sẽ tự có nút copy nhờ doc-script.js.

console.log("Hello, docs template!");

4. Bài tập

  1. Copy thư mục docs-template thành một thư mục mới.
  2. Sửa series.title, series.description và danh sách lessons.
  3. Tạo file bài 01 và kiểm tra liên kết từ trang chủ.

Tổng kết

Template này tách rõ dashboard khóa học và trang bài học, nên có thể dùng lại cho nhiều chủ đề như CCNA, Docker, Golang hoặc Git.
