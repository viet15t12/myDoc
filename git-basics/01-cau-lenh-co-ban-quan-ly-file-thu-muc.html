1. Các câu lệnh cơ bản để quản lý file và thư mục

Tài liệu học Terminal / Command Line cơ bản, tổng hợp từ nội dung video và bổ sung kiến thức thực tế.

Giới thiệu

Tài liệu này giúp bạn nắm các thao tác cơ bản:

  • Điều hướng thư mục
  • Tạo, đọc, ghi, xóa file
  • Tạo và xóa thư mục
  • So sánh file
  • Sử dụng terminal hiệu quả hơn

1. Điều hướng và quản lý thư mục

pwd — Hiển thị thư mục hiện tại

pwd

pwd là viết tắt của Print Working Directory, dùng để xem terminal đang đứng ở thư mục nào.

ls / dir — Liệt kê file và thư mục

ls

Trên Linux/macOS thường dùng ls. Trên Windows Command Prompt thường dùng dir.

Lệnh Ý nghĩa
ls Liệt kê file và thư mục
ls -l Hiển thị chi tiết
ls -a Hiển thị cả file ẩn
ls -la Kết hợp hiển thị chi tiết và file ẩn

cd — Chuyển thư mục

cd Documents

Đi vào thư mục con tên là Documents.

cd ..

Quay lại thư mục cha.

cd
cd ~

Quay về thư mục home của người dùng.

Mẹo: Gõ vài ký tự đầu của tên thư mục rồi nhấn Tab để terminal tự động gợi ý tên đầy đủ.

2. Tạo và quản lý thư mục

mkdir — Tạo thư mục mới

mkdir ten_thu_muc

Tạo nhiều thư mục cùng lúc

mkdir folder1 folder2 folder3

Tạo thư mục lồng nhau

mkdir -p project/src/components

Tùy chọn -p giúp tạo toàn bộ cây thư mục nếu các thư mục cha chưa tồn tại.

3. Quản lý file

touch — Tạo file mới

touch file.txt

Tạo file có khoảng trắng trong tên

touch "ten file.txt"

Tạo nhiều file

touch a.txt b.txt c.txt

4. Ghi nội dung vào file

echo — Xuất nội dung

echo "Hello World"

Ghi đè nội dung vào file

echo "Noi dung moi" > file.txt

Ký hiệu > sẽ ghi đè toàn bộ nội dung cũ trong file.

Thêm nội dung vào cuối file

echo "Dong moi" >> file.txt

Ký hiệu >> dùng để thêm nội dung vào cuối file mà không xóa dữ liệu cũ.

5. Đọc nội dung file

cat — Hiển thị nội dung file

cat file.txt

Hiển thị nhiều file

cat file1.txt file2.txt

Hiển thị kèm số dòng

cat -n file.txt

6. So sánh file

diff — So sánh sự khác nhau giữa hai file

diff file1.txt file2.txt

Lệnh này rất hữu ích khi kiểm tra sự thay đổi giữa hai phiên bản file, đặc biệt khi học Git.

7. Xóa file và thư mục

rm — Xóa file

rm file.txt

Xóa thư mục rỗng

rm -d folder

Xóa thư mục có dữ liệu

rm -r folder
Cảnh báo: Lệnh rm trên Linux thường xóa trực tiếp, không đưa vào Recycle Bin. Cần kiểm tra kỹ trước khi chạy.

Xóa cưỡng bức

rm -rf folder

Trong đó:

  • -r: xóa đệ quy cả thư mục con
  • -f: force, không hỏi xác nhận

8. Mẹo vận hành Terminal

clear — Làm sạch màn hình

clear

Có thể dùng phím tắt:

Ctrl + L

Dùng phím mũi tên để xem lịch sử lệnh

↑  ↓

Phím mũi tên lên/xuống giúp duyệt lại các lệnh đã gõ trước đó.

history — Xem lịch sử command

history

9. Một số lệnh hữu ích khác

cp — Sao chép file

cp file.txt backup.txt

mv — Di chuyển hoặc đổi tên file

mv old.txt new.txt
mv file.txt Documents/

rmdir — Xóa thư mục rỗng

rmdir folder

man — Xem tài liệu lệnh

man ls

Nhấn q để thoát khỏi màn hình tài liệu.

10. Đường dẫn trong Linux

Đường dẫn tuyệt đối

/home/user/Documents/file.txt

Đường dẫn tuyệt đối bắt đầu từ thư mục gốc /.

Đường dẫn tương đối

Documents/file.txt

Đường dẫn tương đối phụ thuộc vào vị trí thư mục hiện tại.

11. Quyền file Linux cơ bản

Xem quyền file

ls -l

Ví dụ:

-rw-r--r--
Ký tự Ý nghĩa
r Read — quyền đọc
w Write — quyền ghi
x Execute — quyền thực thi

12. Bài tập thực hành

Bài tập 1: Tạo cấu trúc thư mục project

project/
├── src/
├── docs/
└── README.md

Gợi ý:

mkdir -p project/src project/docs
touch project/README.md

Bài tập 2: Ghi nội dung vào README.md

echo "# My Project" > README.md

Bài tập 3: So sánh hai file

diff a.txt b.txt

13. Mối liên hệ với Git

Lệnh Terminal Liên quan khi học Git
ls Xem nội dung project/repository
cd Di chuyển vào thư mục repository
mkdir Tạo thư mục project
rm Xóa file khỏi project
cat Đọc nội dung file cấu hình hoặc code
diff Hiểu sự khác biệt giữa các phiên bản file

Tổng kết

Sau khi học xong bài này, bạn cần nắm được:

  • Cách di chuyển giữa các thư mục bằng cd
  • Cách xem nội dung thư mục bằng ls hoặc dir
  • Cách tạo thư mục bằng mkdir
  • Cách tạo file bằng touch
  • Cách ghi nội dung bằng echo, >, >>
  • Cách đọc file bằng cat
  • Cách so sánh file bằng diff
  • Cách xóa file/thư mục bằng rm
